## 摘要

我们研究视觉-语言-动作（VLA）模型如何通过真实世界部署中的强化学习（RL）进行改进。我们提出一种通用方法，**通过优势条件策略进行 RL（RECAP）**，通过优势条件化为 VLA 提供 RL 训练。

我们的方法将异构数据整合到**自我改进**过程中，包括演示、在线策略收集的数据，以及在自主执行期间提供的专家遥操作纠正。RECAP 通过离线 RL 预训练通用 VLA（称为 **π*0.6**），然后可以通过机器人数据收集专门化以在下游任务上实现高性能。

我们展示了使用完整 RECAP 方法训练的 π*0.6 模型可以在真实家庭中折叠衣物、可靠组装盒子，以及使用专业浓缩咖啡机制作浓缩咖啡饮品。在一些最难的任务上，RECAP 将任务吞吐量翻倍以上，任务失败率大约减半。

## 1 引言

"如果你不怕尝试，你能学到的东西令人惊奇。"——罗伯特·海因莱因

**RECAP（RL with Experience and Corrections via Advantage-conditioned Policies）**：
- 整合演示、自主经验和专家纠正
- 通过优势条件化从次优数据中提取改进策略

1. **RECAP 方法**：通过优势条件化训练 VLA 的通用方法
2. **π*0.6 模型**：基于 π0.6 的 RL 适应版本
3. **分布化价值函数**：多任务价值函数预测到成功的步数

## 3 预备知识

### 强化学习

**目标**：最大化累积奖励 J(π) = E[Σ r_t]

**价值函数**：V^π(o_t) = E[Σ_{t'=t}^T r_{t'}]

**优势函数**：A^π(o_t, a_t) = E[Σ r_{t'} + V^π(o_{t+N})] - V^π(o_t)

### 正则化 RL

**目标**：J(π, π_ref) = E[Σ γ^t r_t] - β·E[D(π(·|o) || π_ref(·|o))]

**闭式解**：π̂(a|o) ∝ π_ref(a|o) · exp(A^π_ref(o,a)/β)

## 4 RECAP

### 4A. 分布化价值函数训练

**多任务分布化价值函数**：
```
p_ϕ(V|o_t, ℓ) ∈ Δ^B
```

**训练目标**：
```
min_ϕ E_τ[Σ_{o_t ∈ τ} H(R^B_t(τ), p_ϕ(V|o_t, ℓ))]
```

**关键设计**：
- B=201 个离散化 bin
- 使用蒙特卡洛估计器
- 预测到成功的步数

### 4B. 通过优势条件化的策略提取

**改进策略的闭式解**：
```
π̂(a|o,ℓ) ∝ π_ref(a|o,ℓ) · (π_ref(a|I,o,ℓ) / π_ref(a|o,ℓ))^β
```

**训练目标**：
```
min_θ E[-log π_θ(a_t|o_t,ℓ) - α·log π_θ(a_t|I_t,o_t,ℓ)]
```

**优势指标**：
```
I_t = 1(A^π_ref(o_t,a_t,ℓ) > ε_ℓ)
```

**关键洞察**：
- 类似分类器自由引导（CFG）
- 避免显式表示改进概率
- 使用阈值 ε_ℓ 而非调优 β

### 4C. 方法总结

**数据来源**：
- 演示数据
- 自主推出数据
- 人类纠正数据

**价值函数**：预测到成功的步数，归一化到 (-1, 0)

## 6 实验

### 任务

| 任务 | 描述 | 挑战 |
|------|------|------|
| **浓缩咖啡** | 使用专业机制作咖啡 | 液体操作，多阶段 |
| **盒子组装** | 组装纸板盒子 | 柔性物体，精度 |

### 核心结果

**性能提升**：
- 最难任务：吞吐量翻倍
- 失败率：减半或更多

**长期部署**：
- 浓缩咖啡：连续 13 小时无失败
- 衣物折叠：新家庭中 2 小时不间断
- 盒子组装：工厂真实包装使用
