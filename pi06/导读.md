## 🎯 一句话总结

**提出 RECAP，一种通过优势条件策略使 VLA 模型从经验中学习的方法，整合演示、自主数据和人类纠正，在最难任务上将吞吐量翻倍，失败率减半，实现 13 小时连续制作浓缩咖啡。**

---

## 🔍 研究背景与动机

### 核心问题

**实践造就完美**：VLA 模型需要通过反复练习来掌握技能，不仅利用演示数据，还要利用自主收集的经验数据。

### 核心思想

**RECAP（RL with Experience and Corrections via Advantage-conditioned Policies）**：
- 整合演示、自主经验和专家纠正
- 通过优势条件化从次优数据中提取改进策略

---

## 💡 核心贡献

1. **RECAP 方法**：通过优势条件化训练 VLA 的通用方法
2. **π*0.6 模型**：基于 π0.6 的 RL 适应版本
3. **分布化价值函数**：多任务价值函数预测到成功的步数
4. **实际部署**：13 小时连续浓缩咖啡、2 小时折叠衣物

## 📖 方法详解

### RECAP 方法概述

**三步循环**：
1. **数据收集**：运行 VLA，标注成功/失败，可选人类纠正
2. **价值函数训练**：在所有数据上训练多任务分布化价值函数
3. **优势条件训练**：使用优势指标改进 VLA 策略

**算法流程**：
```
1. 在 D_demo 上训练 V_pre（使用 Eq.1）
2. 在 D_demo 上训练 π_pre（使用 Eq.3 和 V_pre）
3. 用 ℓ 的演示初始化 D_ℓ
4. 从 V_pre 在 D_ℓ 上训练 V^0_ℓ
5. 从 π_pre 在 D_ℓ 上训练 π^0_ℓ
6. for k = 1 to K:
7.   用 π^{k-1}_ℓ 收集数据，加入 D_ℓ
8.   从 V_pre 在 D_ℓ 上训练 V^k_ℓ
9.   从 π_pre 在 D_ℓ 上训练 π^k_ℓ
```

### 4A. 分布化价值函数训练

**多任务分布化价值函数**：
```
p_ϕ(V|o_t, ℓ) ∈ Δ^B
```

**训练目标**：
```
min_ϕ E_τ[Σ_{o_t ∈ τ} H(R^B_t(τ), p_ϕ(V|o_t, ℓ))]
```

**关键设计**：
- B=201 个离散化 bin
- 使用蒙特卡洛估计器
- 预测到成功的步数

### 4B. 通过优势条件化的策略提取

**改进策略的闭式解**：
```
π̂(a|o,ℓ) ∝ π_ref(a|o,ℓ) · (π_ref(a|I,o,ℓ) / π_ref(a|o,ℓ))^β
```

**训练目标**：
```
min_θ E[-log π_θ(a_t|o_t,ℓ) - α·log π_θ(a_t|I_t,o_t,ℓ)]
```

**优势指标**：
```
I_t = 1(A^π_ref(o_t,a_t,ℓ) > ε_ℓ)
```

**关键洞察**：
- 类似分类器自由引导（CFG）
- 避免显式表示改进概率
- 使用阈值 ε_ℓ 而非调优 β

### 4C. 方法总结

**数据来源**：
- 演示数据
- 自主推出数据
- 人类纠正数据

**价值函数**：预测到成功的步数，归一化到 (-1, 0)

## 5 实现、模型和系统细节

### 5A. π0.6 模型

**基于 π0.5 的改进**：
- 更大多机器人数据集
- Gemma 3 4B 基础 VLM
- 860M 参数动作专家

**训练**：知识绝缘（KI）训练过程

### 5B. 从 π0.6 到 π*0.6

**优势条件化**：
- 扩展输入包含优势指标
- "Advantage: positive" 或 "Advantage: negative"
- 出现在训练序列中 ˆℓ 之后、动作之前

### 5C. 奖励定义和价值函数训练

**稀疏奖励**：每个 episode 成功标签

**价值函数**：预测到成功的步数

## 📊 实验设置与主要结果

### 任务

| 任务 | 描述 | 挑战 |

## 🔄 与相关工作的对比

| 方法 | 特点 | 局限 |
|------|------|------|
| PPO 微调 | 直接策略梯度 | 难以扩展到大 VLA |
| 残差策略 | 在冻结 VLA 上学习 | 不是端到端 |
| DPO | 人类偏好优化 | 需要偏好数据 |
| **RECAP** | **优势条件化** | **需要价值函数训练** |

## ⚠️ 局限性与未来方向

- 扩展到更多任务
- 研究自动化奖励设计
