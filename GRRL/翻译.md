## 摘要

我们提出 GR-RL，一种机器人学习框架，将通用视觉-语言-动作（VLA）策略转化为长时程灵巧操作的高能力专家。现有 VLA 策略假设人类演示是最优的，但我们声称在高度灵巧和精密的操作任务中，人类演示充满噪声且次优。

GR-RL 提出多阶段强化训练管道：首先通过强化学习过滤、增强和强化演示。GR-RL 学习视觉-语言条件化的**任务进度**，过滤演示轨迹，仅保留对进度有正向贡献的转移。通过直接应用稀疏奖励的离线 RL，得到的 Q 值可作为鲁棒的进度函数。其次引入**形态对称增强**，大幅提高泛化和性能。最后，通过学习潜在空间噪声预测器执行在线 RL。

据我们所知，GR-RL 是第一个能自主系鞋带的学习策略，将鞋带穿过多个鞋眼，成功率达 83.3%。

## 1 引言

大规模 VLA 模型的出现快速推进了构建通用机器人智能体的目标。然而，通用不等于可靠，当前 VLA 策略在两个基本方面不足：（1）**灵巧精度**——毫米级柔性物体控制仍未解决；（2）**长时程鲁棒性**——误差随步数累积。

### 核心问题

考虑系鞋带任务：（1）需要灵巧处理柔性物体；（2）需要毫米级精度将鞋带穿入鞋眼；（3）需要长时程操作能力应对多样化场景。

**两个关键瓶颈**：
1. **次优人类演示**：人类演示者会减速、犹豫、引入噪声
2. **演示与推理不匹配**：系统级优化导致训练与部署不一致

### 贡献

1. 学习任务进度模型过滤次优数据
2. 形态对称增强提高泛化
3. 在线噪声空间引导对齐训练与部署
4. 首次自主系鞋带（83.3% 成功率）

## 2 GR-RL 模型

**架构**：Mixture-of-Transformer (MoT)，5B 参数

**策略 π_θ**：
- VLM 骨干：Qwen2.5-VL-3B-Instruct
- 动作生成：动作扩散 Transformer (DiT)，流匹配目标
- 仅使用 VLM 后半层的 KV cache 加速推理

**评论家 Q_ϕ**：
- 因果 Transformer
- Q-chunking：为每个动作块预测一串 Q 值
- 分布化 RL：将值作为有界离散分布

## 3 训练方案

### 3.1 用学习的任务进度评估器过滤数据

**稀疏奖励**：
```
r(o_t, l, s_t, a_t) = { γ^{T-t} · I(τ), t > T-k; 0, t ≤ T-k }
```

**进度函数**：Q_ϕ 的分类分布均值 ρ_t

**过滤规则**：如果 ρ_{t:t+k} 中有下降超过阈值 δ，则标记为次优

**失败轨迹增强**：标注重试关键帧，创建额外失败轨迹

### 3.2 带数据增强的模仿学习

**形态对称增强**：
- 水平翻转图像
- 交换左右腕部图像
- 镜像本体感觉和动作
- 翻转语言中的空间描述

### 3.3 用于策略部署对齐的在线引导

**噪声空间探索**：
- 噪声预测器 π_θ'：51.5M 参数
- 预测动作 DiT 的初始噪声 ε_t
- 惩罚偏离正态分布超过阈值 β 的输出

**训练目标**：
```
L(π_θ') = E[-Q_ϕ'(o_t, l, s_t, ε_t) + c · max(||ε_t||²/2 - β, 0)]
L(Q_ϕ') = cross_entropy(Q_ϕ'(..., ε_t), Q_ϕ(..., π_θ(...|ε_t)))
```

**缓冲区设计**：
- 离策略缓冲区：用离线检查点预热
- 在策略缓冲区：仅最近两个检查点
- 不混入遥操作轨迹
