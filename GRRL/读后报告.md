# GRRL (GR-RL) 读后报告

**论文**：GR-RL: Going Dexterous and Precise for Long-Horizon Robotic Manipulation
**作者**：ByteDance Seed 团队
**发表**：arXiv 2025

---

## 一、论文定位

GR-RL 是一篇系统工程论文，提出多阶段强化训练管道，把通用 VLA 策略转化为长时程灵巧操作专家。最大亮点是**首次实现自主系鞋带（83.3% 成功率）**，展示了 RL 增强 VLA 在极端灵巧任务上的潜力。

## 二、核心问题

VLA 模型在灵巧精密任务上的两个瓶颈：

1. **次优人类演示**：人类演示充满噪声、犹豫、不一致（系鞋带时手会抖、会停顿）
2. **训练-部署不匹配**：时序集成等系统优化导致训练与部署不一致

## 三、核心贡献

### 1. 任务进度过滤

用 Q 值作为"进度函数"，过滤掉进度下降的步骤。

```
做法：
  1. 用 TD3+BC 训练评论家 Q_ϕ
  2. Q 值的分类分布均值 ρ_t 作为任务进度
  3. 如果 ρ_{t:t+k} 中有下降超过阈值 δ → 标记为次优
  4. 排除次优转移，只保留"一直在进步"的步骤
```

额外技巧：失败轨迹增强——标注重试关键帧，创建额外失败轨迹。

核心洞察：**数据质量 > 数据数量**，过滤次优数据比增加数据更重要。

### 2. 形态对称增强

利用双臂机器人的天然对称性：
- 水平翻转所有图像
- 交换左右腕部图像
- 镜像对称本体感觉状态和动作
- 翻转语言描述中的空间描述（"左边" ↔ "右边"）

→ 数据量翻倍，且学到了对称性。

### 3. 在线噪声空间引导

问题：动作扩散模型的初始噪声 ε 是随机的，训练时和部署时的噪声分布可能不同。

解决：学习一个噪声预测器 π_θ'（51.5M 参数），预测最优初始噪声 ε_t。

```
训练目标：
  L(π_θ') = E[-Q(ε_t) + c · max(||ε_t||²/2 - β, 0)]
```

惩罚偏离正态分布太远的输出。

### 4. 模型架构

```
策略 π_θ（5B 参数）：
  VLM 骨干：Qwen2.5-VL-3B-Instruct
  动作生成：动作扩散 Transformer (DiT)，流匹配目标
  仅使用 VLM 后半层的 KV cache 加速推理

评论家 Q_ϕ：
  因果 Transformer
  Q-chunking：为每个动作块预测一串 Q 值
  分布化 RL：预测 Q 值的分布，而非单值
```

## 四、实验结果

系鞋带任务：

| 方法 | 成功率 | 提升 |
|------|--------|------|
| GR-3（基础 BC） | 45.7% | - |
| + 进度过滤 | 61.6% | +15.9% |
| + 对称增强 | 72.7% | +11.1% |
| + 在线 RL | **83.3%** | +10.6% |

每个组件都有显著贡献，三者叠加效果最佳。

## 五、关键问答

### Q：这里面的模型还是通过 RL 训练的吗，并没有 VLA？

GR-RL 既有 VLA 也有 RL：
- 基础模型是 VLA（Qwen2.5-VL-3B，5B 参数）
- 用 RL 对 VLA 进行增强（离线过滤 + 在线微调）
- 与 ConRFT 类似，都是 RL 增强 VLA

### Q：数据集中只有人类演示的数据吗？

不只有人类演示：
- 离线数据：人类演示 + 过滤后的次优数据 + 失败轨迹增强
- 在线数据：策略执行产生的新数据
- 缓冲区设计：离策略缓冲区用离线检查点预热，在策略缓冲区仅存储最近两个检查点

### Q：Q-chunking 在 GR-RL 中的作用？

GR-RL 的评论家使用 Q-chunking：为每个动作块预测一串 Q 值。这是从 Q-chunk 论文借鉴的技术。

### Q：分布化评论家是什么？

不是预测单个 Q 值，而是预测 Q 值的概率分布。好处：
- 更鲁棒（分布比回归更稳定）
- 能捕捉不确定性
- 用于任务进度评估（分类分布均值作为进度）

## 七、局限性

1. 目前仅在系鞋带任务上验证
2. 需要 ByteMini-v2 专用硬件
3. 多阶段管道训练流程复杂

## 八、个人启发

1. 数据质量 > 数据数量：过滤次优数据比增加数据更重要
2. 对称性的价值：双臂机器人的形态对称性是免费的增强
3. 噪声空间探索比动作空间探索更适合精密任务
4. 训练-部署对齐是在线 RL 的关键问题
5. 5B 参数的大模型 + RL 是未来趋势
