# 通过人类在环强化学习实现精密灵巧机器人操作

# Precise and Dexterous Robotic Manipulation via Human-in-the-Loop Reinforcement Learning

**作者**：Jianlan Luo, Charles Xu, Jeffrey Wu, Sergey Levine

**机构**：UC Berkeley, Department of Electrical Engineering and Computer Sciences

**发表于**：Science Robotics (2025)

---

## 摘要

强化学习（RL）在实现复杂机器人操作技能的自主获取方面具有巨大潜力，但在真实世界环境中实现这一潜力一直具有挑战性。我们提出了一个人类在环的视觉 RL 系统，在多种灵巧操作任务上展示了令人印象深刻的性能，包括动态操作、精密装配和双臂协调。我们的方法整合了演示和人类纠正、高效 RL 算法以及其他系统级设计选择，学习的策略在仅 1 到 2.5 小时的训练内实现了近完美成功率和快速周期时间。我们展示了我们的方法显著优于模仿学习基线和先前的 RL 方法，平均成功率提升 2 倍，执行速度快 1.8 倍。通过大量实验和分析，我们提供了对我们方法有效性的洞察，展示了它如何学习鲁棒的、自适应的策略用于反应式和预测式控制策略。我们的结果表明，RL 确实可以在实际训练时间内直接在真实世界中学习各种复杂的视觉操作策略。

## 1 引言

操作是机器人学的基础问题之一，在动态灵巧操作任务上实现人类水平性能是该领域长期以来的追求。强化学习（RL）有望实现复杂灵巧机器人技能的自主获取。通过试错学习，有效的 RL 方法原则上能够获取针对部署任务特定物理特征的高度熟练技能。

然而，在真实世界环境中实现这一承诺一直具有挑战性，原因是样本复杂性、假设（如精确奖励函数）和优化稳定性等问题。RL 方法在仿真训练和现有大型真实世界数据集上训练方面已显示出有效性，但开发能够高效获取物理复杂技能的通用视觉方法相对困难。

在本文中，我们开发了一个用于视觉操作的 RL 系统，可以获取各种精确灵巧的机器人技能。我们的系统名为 HIL-SERL（人类在环样本高效机器人强化学习），通过整合多个组件来解决上述挑战。

### 贡献

- 开发了 HIL-SERL 系统，在真实世界中实现了多种精密灵巧操作任务的近完美性能
- 首次使用 RL 在真实世界中实现双臂协调图像输入任务
- 提供了 RL 操作成功的深入实证分析

## 2 相关工作

### 真实世界 RL 算法和系统

真实世界机器人 RL 需要样本高效的算法。相关工作包括离策略 RL、基于模型的 RL 和在策略 RL 的各种变体。与 SERL 最相关的工作相比，我们的方法整合了人类演示和纠正，而 SERL 仅依赖人类演示。

### 灵巧机器人操作

先前工作在插入任务中使用基于模型的方法和末端执行器工具机制，在动态操作任务中使用运动捕捉系统和动态运动原语。我们的方法直接使用像素输入，采用更紧密的感知-动作循环。

## 3 人类在环强化学习系统

### 3.1 预备知识和问题定义

机器人 RL 任务通过 MDP M = {S, A, ρ, P, r, γ} 定义。最优策略 π 最大化累积期望奖励。

核心 RL 算法基于 RLPD（Ball et al., 2023），以其样本效率和整合先验数据的能力。每个训练步，RLPD 在先验数据和在线数据之间等量采样形成训练 batch。

**Q 函数损失**：
```
L_Q(φ) = E[(Q_φ(s,a) - (r(s,a) + γ·E[Q̄_φ(s',a')]))²]
```

**策略损失**：
```
L_π(θ) = -E[E[Q_φ(s,a)] + α·H(π_θ(·|s))]
```

### 3.2 系统概述

系统由三个主要组件组成：

**Actor 进程**：
- 与环境交互，执行当前策略
- 模块化环境：支持多相机、SpaceMouse 遥操作、多机械臂
- 人类可通过 SpaceMouse 干预

**Learner 进程**：
- 从 Demo Buffer 和 RL Buffer 等量采样
- 使用 RLPD 更新策略

**回放缓冲区**：
- Demo Buffer：存储离线人类演示（20-30 个）
- RL Buffer：存储在线策略数据

### 3.3 系统设计选择

**预训练视觉骨干**：使用 ImageNet 预训练的 ResNet-10 处理图像，提供优化稳定性和探索效率。

**奖励函数**：使用稀疏奖励 + 训练的二元分类器，仅在任务完成时给予正奖励。

**下游机器人系统**：
- 相对坐标系表示本体感觉状态（自我中心公式）
- 阻抗控制器确保安全
- 动态任务直接命令前馈力矩

**夹爪控制**：
- 使用单独的评论家网络评估离散抓取动作
- 解决两个独立 MDP：连续动作空间 + 离散动作空间
- 标准 DQN 更新

### 3.4 人类在环强化学习

人类操作员在训练期间监督机器人，必要时提供纠正动作。人类接管控制最多 N 步。干预数据存储在 Demo 和 RL 缓冲区。

**关键洞察**：
- 策略改进时干预频率逐渐降低
- 具体纠正比长时间稀疏干预更有效
- 避免持续长时间干预导致价值函数高估

### 3.5 训练流程

1. 选择相机（腕部 + 侧面）
2. 收集奖励分类器数据（~200 正样本 + ~1000 负样本，~5 分钟）
3. 收集人类演示（20-30 个轨迹）
4. 策略训练（人类干预直到收敛）

## 4 实验结果

### 4.1 实验概述

在 7 个多样化任务上进行实验，涵盖动态对象操作、精密装配、柔性物体操作和多阶段任务。

### 4.2 任务描述

**主板装配**（4 个子任务）：
- RAM Insertion：将 RAM 卡插入插槽
- SSD Assembly：安装 SSD 到主板
- USB Grasp-Insertion：抓取 USB 并插入
- Cable Clipping：将 USB 线缆插入夹子

**IKEA Assembly**（3 个子任务）：
- Side Panel Assembly：组装侧板
- Top Panel Assembly：安装顶板
- Whole Assembly：完整组装

**Car Dashboard Assembly**：双臂协调组装汽车仪表板

**Object Handover**：双臂物体交接

**Timing Belt Assembly**：双臂协作安装正时皮带

**Jenga Whipping**：鞭打抽出积木块

**Object Flipping**：翻转锅中物体

### 4.3 实验结果

**HIL-SERL vs 模仿学习**：

| 任务 | BC 成功率 | HIL-SERL 成功率 | 提升 | 周期时间提升 |
|------|-----------|-----------------|------|-------------|
| RAM Insertion | 29% | 100% | +245% | 1.7x |
| SSD Assembly | 79% | 100% | +27% | 2x |
| USB Grasp-Insertion | 26% | 100% | +285% | 2x |
| Cable Clipping | 95% | 100% | +5% | 1.7x |
| IKEA - Side Panel | 77-79% | 100% | +27-30% | 2.1-2.4x |
| IKEA - Top Panel | 35% | 100% | +186% | 3.7x |
| Car Dashboard | 41% | 100% | +144% | 2.3x |
| Object Handover | 79% | 100% | +27% | 1.2x |
| Timing Belt | 2% | 100% | +4900% | 1.3x |
| Jenga Whipping | 8% | 100% | +1150% | - |
| Object Flipping | 46% | 100% | +117% | 1.03x |
| **平均** | **49.7%** | **100%** | **+101%** | **1.8x** |

**消融实验**：

| 方法 | 平均成功率 |
|------|-----------|
| Diffusion Policy | 34% |
| HG-DAgger | 39% |
| BC | 31% |
| IBRL | 57% |
| Residual RL | 32% |
| DAPG | 33% |
| HIL-SERL（无演示无干预） | 0% |
| HIL-SERL（有演示无干预） | 49% |
| **HIL-SERL（完整）** | **100%** |

### 4.4 鲁棒性结果

训练后的策略展现了出色的零样本鲁棒性：
- RAM Insertion：在主板移动时仍能成功
- Object Handover：夹爪被强制打开后能重试
- Timing Belt：适应皮带任意变形
- Dashboard Assembly：夹爪被打开后能重新抓取
- USB Grasp-Insertion：适应外部干扰和不良抓取

---

## 5 结论

我们展示了通过适当的系统级设计选择，RL 可以在真实世界中有效解决各种灵巧复杂视觉操作任务。我们的系统是首个在真实世界中使用 RL 实现双臂协调图像输入的系统。我们希望这项工作将作为未来基于学习的机器人操作研究的垫脚石。
