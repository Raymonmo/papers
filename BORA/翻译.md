## 摘要

视觉-语言-动作（VLA）模型已成为将视觉-语言理解落地到真实世界机器人操作的有前景范式。然而，灵巧操作对于 VLA 策略仍然具有挑战性，原因是高维手部控制和复合执行误差，这使得真实世界 RL 后训练对于弥合视觉基础动作生成和物理可靠灵巧执行之间的差距至关重要。

然而，高维灵巧探索常常触发时间不一致、样本低效和硬件风险。为解决这些挑战，我们提出 **BORA**，一种专为真实世界灵巧 VLA 模型设计的离线到在线 RL 后训练框架。

在离线阶段，BORA 构建一个评论家，将 VLM 的认知 token 和动作块作为输入。这种设计实现了动作条件价值引导，使评论家能够评估超越视觉上下文的灵巧手部运动。在随后的在线阶段，BORA 冻结 VLA 基础，引入轻量级人类在环（HiL）块级残差适应机制。

在五个复杂真实世界灵巧任务上的广泛评估表明，BORA 显著优于纯模仿学习和传统解耦 RL 基线，在标准设置下平均成功率绝对提升 **33%**，未见物体泛化提升高达 **43%**。

## 1 引言

虽然 VLA 模型已成为通用机器人控制的强大范式，但由于涉及的高度自由度（DoFs），它们在真实世界灵巧操作中面临严重挑战。

### 核心问题

1. **信用分配失败**：生成式动作架构依赖数百步去噪链，RL 梯度噪声严重累积
2. **视觉遮挡**：灵巧操作常涉及严重遮挡，评论家容易过拟合背景伪影
3. **执行偏差**：真实世界部署引入复杂摩擦和接触动力学导致的执行误差

### 贡献

1. **动作条件评论家**：融合连续动作块与 VLM 认知 token
2. **轻量级残差在线适应**：冻结 VLA 基础，人类在环残差 RL
3. **BORA 统一框架**：离线到在线的完整 RL 后训练框架

## 2 相关工作

### 灵巧操作的 VLA 模型

VLA 模型通过动作头将视觉-语言表示适应到机器人控制。灵巧手比平行夹爪提供更大能力。

### VLA 策略的后训练和适应

离线 RL 和模仿微调可重用数据但收益有限。在线 RL 直接优化任务成功但成本高且不稳定。

## 3 方法

### 3.1 离线 RL：动作条件评论家和一致性策略

**一致性策略**：
- 1-3 步去噪生成高保真动作块
- 截断计算图，确保梯度高效回传

**动作条件评论家**：
- 条件化于 VLM 语义 token z_t 和动作块 A_t
- 分解为原子逐步评分：
```
Q_ϕ(s_t, A_t) = [q_ϕ(z_t, a_{t+i}, i)] ∈ R^k
V_ψ(s_t) = [v_ψ(z_t, i)] ∈ R^k
```
- 移位价值引导实现信用传播：
```
y_{t,i} = r_{t+i} + γ(1-d_{t+i})·v_ψ(z_t, i+1), i < k-1
```

**保守策略改进**：
- 块内动作级 GAE 递归
- 有效性掩码裁剪 PPO 代理目标
- 结合 BC 正则化

### 3.2 部署差距桥接：HiL 残差块适应

**残差块演员**：
- 冻结 VLA 基础
- 轻量级 MLP 参数化
- A_final = A_base + λ_res · π_res(s_prop, A_base, z_VLM)

**评论家继承**：
- 离线训练的评论家初始化在线价值函数
- 保守价值引导目标

**干预驱动 RLPD**：
- 1:1 采样比：静态离线数据集 vs 在线缓冲区
- 人类干预数据动态丰富

**人类在环干预**：
- 人类操作员在失败时接管控制
- 线性插值动作平滑
- 非对称干预驱动奖励

## 4 实验

### 5 个真实世界灵巧任务

| 任务 | 描述 | 挑战 |
|------|------|------|
| **Pick the plush toy** | 抓取毛绒玩具 | 灵巧手抓取 |
| **Pick and Place** | 抓取放置 | 精密操作 |
| **Open the box** | 打开盒子 | 接触丰富 |
| **Pull the tissue** | 拉纸巾 | 柔性物体 |
| **Press the button** | 按按钮 | 精密控制 |

### 核心结果

**BORA vs 基线**：
- 标准设置：平均成功率 **+33%**
- 未见物体泛化：**+43%**
- 动作条件评论家优于解耦评论家
- 在线残差适应进一步提升性能
