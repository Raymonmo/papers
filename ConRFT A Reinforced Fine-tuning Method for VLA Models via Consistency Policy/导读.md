## 🎯 一句话总结

**提出 ConRFT，一种基于一致性策略（Consistency Policy）的强化微调方法，通过离线和在线两阶段统一训练目标，在仅 45-90 分钟的在线微调内实现 96.3% 的平均成功率，比监督微调方法提升 144%。**

---

## 🔍 研究背景与动机

### 问题

1. **VLA 模型的局限性**：预训练 VLA 模型在处理真实世界场景复杂性时往往表现不足
2. **监督微调（SFT）的挑战**：
   - 模型性能严重依赖任务特定数据的质量和数量
   - 人类收集的数据集可能包含次优轨迹和不一致的动作
   - 在接触丰富的环境中尤其困难
3. **直接应用 RL 的困难**：
   - VLA 模型需要在真实世界机器人任务中进行直接物理交互
   - 安全和成本约束要求高样本效率和风险感知探索

### 核心思想

借鉴 LLM/VLM 领域中强化学习微调的成功经验，将 RL 引入 VLA 模型微调，通过统一的一致性策略训练目标，结合离线和在线两个阶段实现高效微调。

---

## 💡 核心贡献

1. **ConRFT 框架**：提出基于一致性策略的强化微调新管道，统一离线和在线训练目标
2. **Cal-ConRFT（离线阶段）**：整合行为克隆和 Q 学习，从少量演示中高效提取策略和价值函数
3. **HIL-ConRFT（在线阶段）**：保持相同损失结构实现快速策略适应，利用人类干预确保安全探索

---

## 📖 方法详解

### 问题建模

将每个机器人任务建模为马尔可夫决策过程（MDP）：M = (S, A, P, r, ρ, γ)

- 预训练 VLA 模型 π_ϕ^pre 编码视觉输入和语言指令
- 目标：在目标任务上用少量标注演示微调模型

### 阶段一：离线微调 Cal-ConRFT

**核心挑战**：预训练 VLA 模型通常缺乏对新机器人配置的零样本泛化能力

**方法**：
- 使用少量预收集离线数据（20-30 个演示）训练
- 以预训练 VLA 模型初始化策略
- 基于 Cal-QL（校准 Q 学习）作为基础离线 RL 方法

**统一训练目标**：

```
L_π^offline(ψ) = β·L_BC^π + η·L_Q^π
```

- **BC 损失**：L_BC = E[d(f_ψ(a + k_m·z, k_m|E(s)), a)]，直接最小化策略动作与演示动作的差异
- **Q 损失**：L_Q = -E[Q(s, a)]，通过 Q 函数引导策略优化
- **一致性策略**：将扩散范围 [ε, K] 离散化为 M 个子区间

**关键创新**：
- 一致性策略作为动作头，处理不一致和次优演示
- 相比扩散动作头，计算更轻量
- BC 损失提供额外监督信号，帮助在少量演示下学习有效策略

### 阶段二：在线微调 HIL-ConRFT

**核心挑战**：离线阶段性能受限于预收集演示的范围和质量

**方法**：
- VLA 模型通过一致性策略与真实世界环境交互进行在线微调
- 保留离线阶段的演示缓冲区 D，新增回放缓冲区 R
- 使用对称采样：每个 batch 从两个缓冲区等量采样

**在线训练目标**：

```
L_π^online(ψ) = β·L_BC^π + η·L_Q^π
```

与离线阶段结构相同，实现快速适应。

**人类在环学习（HIL）**：
- 人类操作员通过遥操作工具（如 SpaceMouse）干预机器人策略
- 人类纠正添加到演示缓冲区 D
- 确保安全探索和加速策略收敛

**BC 损失保留的原因**：
1. 确保策略继续与演示数据对齐，防止性能崩溃
2. 在探索中提供稳定效果，防止策略偏离太远

## 📊 实验设置与主要结果

### 实验平台
- **机器人**：7 自由度 Franka Emika 机械臂
- **基础模型**：Octo-small
- **状态观测**：腕部相机 RGB (128×128) + 侧面相机 RGB (256×256) + 本体感觉状态
- **动作空间**：6维末端执行器增量姿态 或 7维（含夹爪控制）

### 8 个真实世界任务

| 任务 | 描述 | 特点 |
|------|------|------|
| Pick Banana | 抓取香蕉放到盘子上 | 基础抓取放置 |
| Put Spoon | 放勺子到毛巾上 | 精细抓取 |
| Open Drawer | 打开抽屉 | 接触丰富 |
| Pick Bread | 拿面包 | 精细操作 |
| Open Toaster | 打开烤面包机 | 接触丰富 |
| Put Bread | 放面包到盘子上 | 精细放置 |
| Insert Wheel | 安装椅子轮子 | 高精度装配 |
| Hang Chinese Knot | 挂中国结 | 柔性物体操作 |

### 核心结果

| 方法 | 平均成功率 | 平均 episode 长度 |
|------|-----------|-------------------|
| SFT（离线） | 39.4% | 57.5 步 |
| Cal-ConRFT（离线） | 39.4% | 57.5 步 |
| HG-DAgger（在线） | 65% | 56.3 步 |
| PA-RL（在线） | 71.3% | 51.1 步 |
| **HIL-ConRFT（在线）** | **96.3%** | **30.7 步** |

**关键发现**：
1. **ConRFT 优于监督方法**：平均成功率提升 144%，episode 长度缩短 1.9 倍
2. **微调 VLA 优于从头训练**：HIL-SERL 从头训练仅达 31.9%，HIL-ConRFT 达 96.3%
3. **Cal-ConRFT 提供更好初始化**：引入 Q 损失使在线阶段适应更快
4. **跨 VLA 模型通用性**：在 RoboVLM 上测试（Kosmos-2 和 PaliGemma），成功率均达 100%

## 🔄 与相关工作的对比

| 方法 | 特点 | 局限 |
|------|------|------|
| SFT | 纯监督模仿学习 | 受限于数据质量和数量 |
| HG-DAgger | 人类纠正 + 监督学习 | 人类纠正不一致导致性能不稳定 |
| PA-RL | 策略无关 Q 函数优化 | 保守策略限制探索，复杂任务效果差 |
| HIL-SERL | 从头 RL + 人类干预 | 训练时间长，安全风险高 |
| **ConRFT** | 统一一致性目标 + 两阶段 | 需要任务特定奖励分类器 |

## ⚠️ 局限性与未来方向

### 局限性
1. **奖励工程敏感性**：使用任务特定二元分类器计算奖励，可能存在 reward hacking
2. **编码器和 Transformer 骨干冻结**：仅微调动作头，限制了感知和表示模块的优化
3. **任务特定性**：高度特化的策略，泛化到新环境的能力有限

### 未来方向
- 引入多任务密集奖励信号
- 使用参数高效微调（如 LoRA）更新部分冻结组件
- 探索更通用的奖励设计

---

## 💬 个人评价与启发

### 优点
1. **实用性强**：45-90 分钟即可完成在线微调，适合真实世界部署
2. **统一框架**：离线和在线阶段使用相同的训练目标，设计优雅
3. **安全性保障**：人类在环机制确保探索安全
4. **样本效率高**：仅需 20-30 个初始演示
5. **通用性好**：可应用于不同 VLA 模型架构

### 启发
1. **RL + VLA 的结合是可行的**：通过合适的框架设计，RL 可以有效增强 VLA 模型
2. **一致性策略的优势**：相比扩散策略更轻量，同时能处理不一致数据
3. **两阶段设计的重要性**：离线初始化为在线探索提供安全起点
4. **人类干预的价值**：不仅确保安全，还加速收敛

## 📚 关键术语表

| 英文术语 | 中文翻译 | 解释 |
|----------|----------|------|
| Consistency Policy | 一致性策略 | 基于扩散模型的策略，学习从随机动作到专家动作分布的映射 |
| Reinforced Fine-tuning (RFT) | 强化微调 | 使用强化学习对预训练模型进行微调 |
| Behavior Cloning (BC) | 行为克隆 | 通过模仿专家演示学习策略 |
| Q-Learning | Q 学习 | 通过学习动作价值函数优化策略 |
| Cal-QL | 校准 Q 学习 | 一种离线 RL 方法，对 OOD 动作进行保守惩罚 |
| Human-in-the-Loop (HIL) | 人类在环 | 人类参与训练过程提供纠正 |
| Offline-to-Online | 离线到在线 | 先用离线数据预训练，再在线微调的范式 |
