# ConRFT：基于一致性策略的 VLA 模型强化微调方法

# ConRFT: A Reinforced Fine-tuning Method for VLA Models via Consistency Policy

**作者**：Yuhui Chen, Shuai Tian, Shugao Liu, Yingting Zhou, Haoran Li, Dongbin Zhao

**机构**：中国科学院自动化研究所 SKL-MAIS; 中国科学院大学人工智能学院

**arXiv**: 2502.05450v2 [cs.RO] 2025年4月14日

---

## 摘要

视觉-语言-动作（VLA）模型在真实世界机器人操作中展现了巨大潜力。然而，通过监督学习微调这些模型由于有限、不一致的演示而难以实现鲁棒性能，特别是在接触丰富的环境中。在本文中，我们提出了一种 VLA 模型的强化微调方法，称为 ConRFT，它由离线和在线微调组成，使用统一的基于一致性的训练目标来应对这些挑战。在离线阶段，我们的方法整合行为克隆和 Q 学习，有效地从少量演示中提取策略并稳定价值估计。在在线阶段，VLA 模型通过一致性策略进一步微调，利用人类干预确保安全探索和高样本效率。我们在八个多样化的真实世界操作任务上评估了我们的方法。它在 45-90 分钟的在线微调内实现了 96.3% 的平均成功率，比先前的监督方法在成功率上提升了 144%，episode 长度缩短了 1.9 倍。这项工作突出了整合强化学习来增强 VLA 模型在真实世界机器人应用中性能的潜力。

## 1 引言

使用视觉-语言-动作（VLA）模型训练通用机器人策略的最新进展在理解和执行各种操作任务方面展现了卓越的能力。这些成功主要归功于大规模模仿式预训练和与机器人动作的对齐。然而，预训练策略在处理真实世界场景的复杂性时往往表现不足。使用领域特定数据进行微调对于优化下游任务的模型性能至关重要。虽然使用人类遥操作数据对 VLA 模型进行监督微调（SFT）仍然是主要的适应方法，但这一过程面临重大挑战：模型的性能严重依赖于任务特定数据的质量和数量。

大语言模型（LLM）和视觉语言模型（VLM）的重大进展突出了强化学习作为弥合策略能力与人类偏好之间差距的强大工具的价值。然而，将这些见解扩展到 VLA 模型带来了独特的挑战，因为与 LLM 不同，VLA 模型需要在真实世界机器人任务中进行直接物理交互。

为了利用基于 RL 的技术通过在线交互数据高效微调 VLA 模型，我们提出了一种强化微调（RFT）方法，由离线和在线阶段组成，使用统一的基于一致性的训练目标。

### 贡献

1. 提出 **ConRFT**（基于一致性的强化微调）方法，统一离线和在线微调的训练目标
2. 提出 **Cal-ConRFT**（离线阶段），整合离线 RL 与基于一致性的行为克隆损失
3. 提出 **HIL-ConRFT**（在线阶段），保持相同损失结构实现快速策略适应，利用人类干预确保安全探索

## 2 相关工作

### 2.1 大模型的强化微调

RL 已被广泛用于微调 LLM 和 VLM。早期工作主要关注结合人类反馈的 RL，或整合任务特定奖励。然而，这些方法通常在自生成的合成环境中学习，而非真实世界环境。

### 2.2 真实世界 RL 系统

真实世界机器人 RL 系统需要样本高效且灵活的算法。与从头训练的先前方法不同，我们的工作专注于利用预训练 VLA 模型提供高质量策略初始化。

### 2.3 离线到在线方法

离线到在线 RL 旨在利用离线数据集初始化策略，然后通过在线交互微调。大多数方法假设大规模、多样化的数据集可用，这在真实世界部署中很少满足。

---

## 3 问题设置与预备知识

我们专注于为下游任务微调预训练 VLA 模型。假设访问预训练 VLA 模型 π_ϕ^pre，它编码视觉输入（如 RGB 图像）和语言指令的高层表示。

在监督微调（SFT）中，我们旨在使用少量标注演示将 ϕ_pre 适应到目标任务上的 ϕ，同时保持模型的通用特征提取能力。

由于演示通常有限、不一致和次优，SFT 在真实世界接触丰富的机器人任务中表现挣扎。我们将每个机器人任务建模为马尔可夫决策过程（MDP）：M = (S, A, P, r, ρ, γ)。

## 4 方法

### 4.1 阶段一：Cal-ConRFT 离线微调

预训练 VLA 模型通常缺乏对新机器人配置的零样本泛化能力。在离线阶段，我们专注于使用少量预收集离线数据集（20-30 个演示）训练策略。

**基础方法：Cal-QL（校准 Q 学习）**

Cal-QL 通过减少时序差分（TD）误差和额外正则器在预收集数据集上训练 Q 函数。该正则器在 OOD 动作的 Q 值超过参考策略价值 V^μ(s) 时进行惩罚：

```
L_Q^offline(θ) = α·E[max(Q_θ(s,a), V^μ(s))] - E[Q_θ(s,a)] + ½·E[(Q_θ(s,a) - B^π Q_θ(s,a))²]
```

**Cal-ConRFT 的统一训练目标**

结合 BC 损失与 Q 引导，在一致性策略目标下：

```
L_π^offline(ψ) = β·L_BC^π + η·L_Q^π
```

其中：
- BC 损失：L_BC = E[d(f_ψ(a + k_m·z, k_m|E(s)), a)]，d 为欧几里得距离
- Q 损失：L_Q = -E[Q(s, a)]
- β 和 η 为平衡超参数

**一致性策略**：将扩散范围 [ε, K] 离散化为 M 个子区间，边界 k_1 = ε ≤ k_2 ≤ ... ≤ k_m = K，ε = 0.002。

### 4.2 阶段二：HIL-ConRFT 在线微调

离线阶段从少量演示数据提供初始策略，但性能受限。在线阶段，VLA 模型通过一致性策略与真实世界环境交互进一步微调。

**在线 Q 损失**：

```
L_Q^online(θ) = E[(Q_θ(s,a) - B^π Q(s,a))²]
```

**在线训练目标**：

```
L_π^online(ψ) = β·L_BC^π + η·L_Q^π
```

与离线阶段结构相同，实现快速适应。通常在线阶段减小 BC 损失权重 β，增大 Q 损失权重 η。

**人类在环学习（HIL）**：
- 人类操作员通过遥操作工具（如 SpaceMouse）在探索过程中提供纠正动作
- 人类纠正添加到演示缓冲区 D
- 确保安全探索并加速策略收敛

**保留 BC 损失的原因**：
1. 确保策略继续与演示数据对齐，防止性能崩溃
2. 在 RL 探索中提供稳定效果

## 5 实验与结果

### 5.1 实验概述

在 8 个多样化操作任务上评估，使用 Octo-small 模型和 7 自由度 Franka Emika 机械臂。

**状态观测**：腕部相机 RGB (128×128) + 侧面相机 RGB (256×256) + 本体感觉状态

**动作空间**：6维末端执行器增量姿态 或 7维（含夹爪控制）

**8 个任务**：
- Pick Banana：抓取香蕉放到盘子上
- Put Spoon：放勺子到毛巾上
- Open Drawer：打开抽屉
- Pick Bread：拿面包
- Open Toaster：打开烤面包机
- Put Bread：放面包到盘子上
- Insert Wheel：安装椅子轮子
- Hang Chinese Knot：挂中国结

### 5.2 实验结果

**ConRFT 优于监督方法**：

| 方法 | 平均成功率 | 平均 episode 长度 |
|------|-----------|-------------------|
| SFT（离线） | 39.4% | 57.5 步 |
| Cal-ConRFT（离线） | 39.4% | 57.5 步 |
| HG-DAgger（在线） | 65% (+65%) | 56.3 步 (1.1x) |
| PA-RL（在线） | 71.3% (+81%) | 51.1 步 (1.2x) |
| **HIL-ConRFT（在线）** | **96.3% (+144%)** | **30.7 步 (1.9x)****

**微调 VLA 优于从头训练**：

| 方法 | 平均成功率 |
|------|-----------|
| HIL-SERL（从头训练） | 31.9% |
| **HIL-ConRFT（微调 VLA）** | **96.3%** |

**分析**：

1. **为什么从 Cal-ConRFT 微调而非 SFT 或 Cal-QL？**
   - SFT 仅依赖模仿学习，需要大量在线微调处理未覆盖状态
   - 仅 Cal-QL 在少量演示下不足（0% 成功率）
   - Cal-ConRFT 结合两者优势

2. **增加演示数量是否提升 SFT 性能？**
   - 使用 150 个演示的 DP 和 SFT 仍不及 ConRFT（20 个演示 + 在线微调）
   - 简单增加人类演示不一定保证更高性能

3. **跨 VLA 模型通用性**：
   - 在 RoboVLM 上测试（Kosmos-2 和 PaliGemma），成功率从 51-53% 提升至 100%

## 6 局限性

### 6.1 奖励工程敏感性

使用任务特定二元分类器计算奖励，分类器训练数据与 RL 探索产生的状态-动作分布之间的分布偏移可能导致 reward hacking。

### 6.2 冻结编码器和 Transformer 骨干

当前实现仅微调动作头网络，保持视觉编码器和 Transformer 骨干冻结。虽然提升了实时性能，但限制了策略在在线训练中优化感知和表示模块的能力。

---

## 7 结论

我们提出了两阶段方法 ConRFT，用于在真实世界机器人应用中强化微调 VLA 模型。首先使用少量演示进行离线微调（Cal-ConRFT），通过统一训练目标（整合 Q 损失和 BC 损失）初始化可靠的策略和价值函数。然后在在线微调阶段（HIL-ConRFT）利用任务特定奖励和人类干预，通过一致性策略微调 VLA 模型。在八个多样化真实世界任务上的实验证明了我们方法在成功率、样本效率和 episode 长度方面的优势。
