# 通过残差 RL 数据生成实现自改进视觉-语言-动作模型

# Self-Improving Vision-Language-Action Models with Data Generation via Residual RL

**作者**：Wenli Xiao, Haotian Lin, Andy Peng, Haoru Xue, Tairan He, Yuqi Xie, Fengyuan Hu, Jimmy Wu, Zhengyi Luo, Linxi "Jim" Fan, Guanya Shi, Yuke Zhu

**机构**：NVIDIA, CMU, UC Berkeley, UT Austin

**arXiv**: 2511.00091v1 [cs.CV] 2025年10月30日

## 摘要

监督微调（SFT）已成为大型视觉-语言-动作（VLA）模型的事实后训练策略，但其对昂贵人类演示的依赖限制了可扩展性和泛化。我们提出了**探测、学习、蒸馏（PLD）**，一个三阶段即插即用框架，通过残差强化学习（RL）和分布感知数据收集改进 VLA。

在**阶段 1（专家获取）**中，我们冻结 VLA 骨干并通过离策略 RL 训练轻量级残差动作器。这些专家在基础策略失败的状态接管，从而探测 VLA 通用策略的失败区域。在**阶段 2（数据收集）**中，我们采用混合推出方案，将残差干预偏向基础策略频繁访问的状态，使收集的轨迹与通用策略的部署分布对齐，同时捕获恢复行为。在**阶段 3（微调）**中，这些精心策划的轨迹通过标准 SFT 蒸馏回通用策略，适用于流匹配和自回归动作头。

我们在多样化设置中评估 PLD：在 LIBERO 基准上达到近乎饱和的 99% 任务成功率，在 SimplerEnv 中实现超过 50% 的性能提升，在真实世界 Franka 机械臂和 YAM 机械臂灵巧操作任务上展示 100% 成功率。

## 1 引言

监督微调（SFT）已成为大语言模型（LLM）的标准后训练范式。受这些成功的启发，相同的方法现在被应用于机器人基础模型，特别是视觉-语言-动作（VLA）策略。

然而，将这种范式从语言转移到机器人学是一个独特的挑战。收集高质量的机器人演示既昂贵又费力。即使有这样的数据，它们通常通过遥操作管道收集，与部署的 VLA 策略**解耦**，留下关键的覆盖空白。

这些挑战提出了以下问题：**VLA 模型能否使用 RL 策划的数据以最少的人力自我改进？**

### 核心观察

数据收集**不应**与基础策略无关：数据收集策略和通用策略必须**交互**，以便探索利用通用策略的先验知识，收集的数据保持与其轨迹分布对齐。

### 贡献

1. **自主后训练方案**：使 VLA 模型无需额外人类演示即可自主改进
2. **RL 生成数据的系统研究**：分析自动数据收集对 SFT 最有益的关键组件
3. **全面实证验证**：在仿真和真实机器人上进行大规模消融实验

## 2 预备知识

### 2.1 任务形式化

研究语言条件操作，使用稀疏二元奖励和 VLA 模型作为基础策略类。策略消耗 (o_t, g) 并输出 7 自由度动作。

### 2.2 监督微调

给定 VLA 策略和演示数据集 D，SFT 通过最大化条件动作似然适应策略。损失实例化取决于动作头架构：
- 自回归/token 头：序列 NLL 损失
- 扩散头：分数匹配 MSE 损失
- 流匹配头：L2 流匹配损失

### 2.3 目标条件 RL

将连续控制建模为 MDP，目标条件设置中每个任务由目标变量 g 指定。使用稀疏二元奖励。

## 3 方法

### 3.1 通过策略先验热启动实现数据高效 RL

基于先前数据高效 RL 的成功，考虑离策略 actor-critic 框架，维护两个独立的缓冲区用于离线和在线经验回放。

**残差动作模块**：
- 条件化于基础策略动作：π_δ(·|s, a_b)
- 使用混合策略：ā = a_b + a_δ
- 通过调度器控制探索范围：a_δ ∈ [-ξ, ξ]

**Q 函数学习**：
```
Q̄_π(s_t, ā_t) ← r(s,a) + γ·E[Q̄_π^target(s_{t+1}, ā_{t+1})]
```

**设计选择**：
- 基础策略为探索提供有用的初始化
- 残差高斯策略易于通过任何现成的离策略 RL 算法训练
- 不显式强制行为约束

### 3.2 为可扩展数据生成引导 RL 专家

**问题**：纯专家数据分布狭窄，可能忽略 OOD 和失败状态

**混合数据收集方案**：
1. 基础策略推出随机步数（基础策略探测）
2. 残差 RL 专家接管，生成恢复行为

**数据组成**：
```
τ_demo = {(s_1, a_{b,1}), ..., (s_{t-1}, a_{b,t-1})} ∪ {(s_t, a_{b,t} + ā_t), ...}
```

**关键洞察**：
- 增加探测长度产生更长的 episode 和更大的多样性
- 更广泛的数据支持导致更好的微调性能

## 4 实验

### 4.1 RL 专家学习的有效性和效率

在 LIBERO-90 的 8 个任务上比较 PLD 与基线方法：
- WSRL：仅离线初始化
- RLPD：无基础策略指导

**结果**：PLD 在所有任务上大幅优于基线，在每个报告的任务上达到超过 95% 性能。

### 4.2 分布内性能

**LIBERO 基准**：

| 模型 | Spatial | Object | Goal | 平均 |
|------|---------|--------|------|------|
| π0 Baseline | 95.2% | 97.6% | 87.4% | 93.4% |
| **π0 + PLD** | **97.7%** | **98.5%** | **95.3%** | **97.2%** |
| OpenVLA Baseline | 92.9% | 99.1% | 83.25% | 91.8% |
| **OpenVLA + PLD** | **99.5%** | **99.1%** | **98.9%** | **99.2%** |

**关键发现**：
- PLD 数据在所有套件和两种架构上一致提升
- 更大的 PLD 数据集单调改进分布内成功率
- 蒸馏后的通用策略显著超越平均专家

### 4.3 分布外泛化

**SimplerEnv**：超过 50% 性能提升

**短到长泛化**：π0 在 LIBERO-90 上微调，在 LIBERO-10 长时程任务上一次性评估

### 4.4 真实世界实验

**Franka 机械臂**：
- 多样化对象抓取和销钉插入
- >99% 成功率

**YAM 双臂**：
- GPU 插拔任务
- 100% 成功率
- 连续 1 小时无人干预运行

### 4.5 消融研究

**数据源对比**：
- PLD 数据 > 人类数据 > RL 专家数据 > 基础策略推出数据

**关键发现**：
1. 残差策略探测和分布感知回放是关键
2. 增加探测长度提升数据多样性
3. PLD 数据在分布内和分布外任务上都有效
4. 少量任务训练（10%）即可实现零样本迁移

---

## 5 结论

我们提出了 PLD，一个三阶段后训练管道，使 VLA 模型能够通过 RL 生成的数据自主改进。通过冻结 VLA 骨干训练轻量级残差策略，使用混合推出方案收集部署对齐的数据，然后通过 SFT 蒸馏回通用策略，PLD 在 LIBERO 基准上达到 99% 成功率，在真实世界任务上达到 100%。
