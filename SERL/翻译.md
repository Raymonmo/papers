## 摘要

近年来，机器人强化学习（RL）领域取得了显著进展，使得能够处理复杂图像观测、在真实世界训练并整合辅助数据（如演示和先验经验）的方法成为可能。然而，尽管有这些进展，机器人 RL 仍然难以使用。实践者公认，这些算法的具体实现细节往往与算法选择同等重要（甚至更重要）。

我们提出，机器人 RL 广泛采用的一个重大挑战是此类方法的相对不可访问性。为解决这一挑战，我们开发了一个精心实现的库，包含样本高效离策略深度 RL 方法，以及计算奖励和重置环境的方法、广泛采用机器人的高质量控制器，以及多个挑战性示例任务。

也许令人惊讶的是，我们发现我们的实现可以实现非常高效的学习，在平均 25 到 50 分钟的训练内获取 PCB 板装配、线缆布线和物体搬运的策略。这些策略达到完美或近完美成功率，即使在扰动下也极端鲁棒，并展现涌现的恢复和纠正行为。

## 4 真实世界中的样本高效机器人 RL

### 4.1 核心 RL 算法：RLPD

**RLPD 的三个关键修改**：
1. **高 UTD 比训练**：每个时间步多次梯度更新
2. **对称采样**：50% 先验数据 + 50% 在线数据
3. **层归一化正则化**：允许更高 UTD 比

**Q 函数损失**：
```
L_Q(φ) = E[(Q_φ(s,a) - (r(s,a) + γ·E[Q̄_φ(s',a')]))²]
```

**策略损失**：
```
L_π(θ) = -E[E[Q_φ(s,a)] + α·H(π_θ(·|s))]
```

### 4.2 使用分类器的奖励规范

**二元分类器**：r(s) = log p(e|s)

**VICE（对抗训练）**：
- 将策略访问的所有状态添加到分类器训练集（负标签）
- 每次迭代后更新分类器
- 类似 GAN：策略是生成器，分类器是判别器

**手工规范**：当机器人状态足以评估成功时

### 4.3 前向-后向控制器的无重置训练

**两个策略同时训练**：
- **前向策略**：执行任务
- **后向策略**：返回初始状态

**优势**：消除人工重置需求

### 4.4 软件组件

**环境适配器**：兼容 Gym 环境

**Actor 和 Learner 节点**：并行训练和执行，解耦动作推理和策略更新

### 4.5 接触丰富任务的阻抗控制器

**阻抗控制目标**：
```
F = k_p · e + k_d · ė + F_ff + F_cor
```

**参考限制机制**：
- 约束 e 使得 |e| ≤ Δ
- 在实时层裁剪参考，而非限制 RL 策略动作
- 接触时限制交互力，自由空间允许快速移动

**关键洞察**：
- 直接裁剪 RL 动作会导致学习极其缓慢
- 在实时层裁参考可以限制接触力而不影响自由空间移动

### 4.6 相对观测和动作框架

**相对坐标系**：目标相对于机器人基座固定

**训练时模拟移动目标**：无需物理移动

## 5 实验

### 任务

| 任务 | 描述 | 挑战 |
|------|------|------|
| **PCB 板装配** | 将 PCB 板插入插座 | 精密接触，脆弱引脚 |
| **线缆布线** | 将线缆布入路径 | 柔性物体 |
| **物体搬运** | 将物体从一个箱子移到另一个 | 无重置训练 |

### 核心结果

| 任务 | 训练时间 | 成功率 |
|------|----------|--------|
| PCB 板装配 | 25-50 分钟 | 完美/近完美 |
| 线缆布线 | 25-50 分钟 | 完美/近完美 |
| 物体搬运 | 25-50 分钟 | 完美/近完美 |

**关键发现**：
- 25-50 分钟训练即可达到完美成功率
- 即使在扰动下也极端鲁棒
- 展现涌现的恢复和纠正行为

## 6 结论

我们提出了 SERL，一个开源软件框架，旨在使真实世界机器人 RL 可访问。SERL 包含样本高效 RL 算法、奖励规范方法、重置自动化、通用适配器和阻抗控制器。我们的实验表明，精心实现的现有技术结合精心设计的控制器和精心选择的组件，可以提供对真实世界使用足够高效的系统。
