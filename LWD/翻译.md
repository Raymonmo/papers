# 部署时学习：通用机器人策略的舰队级强化学习

# Learning while Deploying: Fleet-Scale Reinforcement Learning for Generalist Robot Policies

**作者**：Yi Wang, Xinchen Li, Pengwei Xie, Pu Yang, Buqing Nie, Yunuo Cai, Qinglin Zhang, Chendi Qu, Jeffrey Wu, Jianheng Song, Xinlin Ren, Jingshun Huang, Mingjie Pan, Siyuan Feng, Zhi Chen, Jianlan Luo

**机构**：Shanghai Innovation Institute, AGIBOT Finch, Columbia University

**项目网站**：https://finch.agibot.com/research/lwd

## 摘要

通用机器人策略越来越多地从大规模预训练中受益，但仅离线数据不足以实现鲁棒的真实世界部署。部署的机器人遇到分布偏移、长尾失败、任务变化和人类纠正机会，固定演示数据集无法完全捕获。

我们提出**部署时学习（LWD）**，一种舰队级离线到在线强化学习框架，用于通用视觉-语言-动作（VLA）策略的持续后训练。从预训练 VLA 策略开始，LWD 通过使用跨机器人舰队收集的自主推出和人类干预，在部署、共享物理体验、策略改进和重新部署之间关闭循环。

为稳定从异构、稀疏奖励舰队数据中学习，LWD 结合**分布化隐式价值学习（DIVL）**进行鲁棒价值估计，通过**伴随匹配 Q 学习（QAM）**在基于流的 VLA 动作生成器中进行策略提取。

我们在 16 台双臂机器人、8 个真实世界操作任务上验证 LWD，包括语义杂货补货和 3-5 分钟长时程任务。单个通用策略随舰队经验积累而改进，达到 95% 平均成功率，长时程任务提升最大。

## 1 引言

在真实世界部署通用机器人需要高性能通用策略。最近的 VLA 策略从大型离线机器人数据集获取广泛能力提供了强大基础。然而，仅离线预训练不能使策略准备好部署。

**核心观点**：部署不是训练的终点，而是持续策略改进的来源。

**数据飞轮**：部署机器人生成经验 → 共享策略改进 → 改进策略重新部署 → 收集更广泛经验

### 贡献

1. **LWD 框架**：舰队级离线到在线 RL 系统
2. **DIVL**：分布化隐式价值学习
3. **QAM**：伴随匹配策略提取
4. **统一训练目标**：离线和在线阶段一致

## 2 相关工作

### 机器人通用策略后训练

**离线后训练**：π*0.6 结合离线价值学习和迭代 RL。RLDG 使用专家 RL 生成数据蒸馏。

**在线 RL 后训练**：VLA-RL、RIPT 在仿真中实现强改进，但依赖在策略数据。

**离线到在线**：先前方法学习任务特定策略，限制泛化。

### 离线到在线 RL

**DSRL**：在潜在噪声空间适应扩散策略。

**QAM**：使用评论家梯度通过伴随匹配改进流策略。

### 大规模机器人 RL 系统

QT-Opt 等系统展示离策略 RL 可以扩展。SOP Pan 形式化可扩展在线后训练的系统基板。

## 3 预备知识

### 3A. 问题设置和符号

**MDP**：M = (S, A, T, r, γ)

**状态**：s = (o, ℓ_k)，观测 + 语言指令

**稀疏二元奖励**：r = 1 仅在成功终止时

**动作块**：a_t = a_{t:t+H} ~ π_θ(·|s_t)

### 3B. 隐式 Q 学习（IQL）

**标量状态价值函数**：
```
L_V^IQL(ψ) = E[ρ_{τ,2}(Q̄_ϕ(s_t, a_t) - V_ψ^IQL(s_t))]
```

**非对称引导**：τ > 1/2 时偏向高价值数据集动作

### 3C. 流匹配和伴随匹配 Q 学习

**流匹配**：表示为时间依赖向量场的生成策略

**QAM**：KL 正则化改进目标
```
π*(a|s) ∝ π_β(a|s) · exp(Q_ϕ(s,a)/λ)
```

## 4 部署时学习

### 4A. 分布化隐式价值学习（DIVL）

**分布化价值模型**：
```
p_ψ(v|s_t) = P(v = Q_ϕ(s_t, a_t) | a_t ~ D(·|s_t))
```

**训练目标**：
```
L_V(ψ) = E[-log p_ψ(Q̄_ϕ(s_t, a_t) | s_t)]
```

**分位数统计**：
```
Quant_τ(V_ψ(s_t)) = inf{v : F_ψ(v|s_t) ≥ τ}
```

**TD 目标**：
```
y_Q = r_t + γ^H · Quant_τ(V_ψ(s_{t+H}))
```

**自适应 τ**：
```
τ(s_{t+H}) = clip(τ_base - α·H(s_{t+H}), τ_min, τ_max)
```

**命题 1**：分布化两步过程与直接非对称价值回归有相同最优解

### 4B. 通过 QAM 的策略提取

**伴随匹配损失**：
```
L_QAM(θ) = E[∫₀¹ ||f_δ(s, a_w, w) - σ_w + σ_w·g̃_w/2||² dw]
```

**关键洞察**：
- 避免通过完整多步去噪过程反向传播
- 将轨迹级优化转化为沿参考流的局部回归

### 4C. 离线到在线 RL 训练管道

**阶段 1：离线 RL 预训练**
- 数据来源：演示、推出、游戏数据
- 使用 n 步 chunk 级 TD 目标冷启动

**阶段 2：持续在线后训练**
- 部署当前策略到机器人舰队
- 混合回放 B_off ∪ B_on 更新
- 定期重新部署更新的策略

**数据飞轮**：部署 → 收集 → 更新 → 重新部署
