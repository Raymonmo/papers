## 摘要

我们提出 **Q-chunking**，一种简单而有效的配方，用于改进强化学习（RL）算法处理长时程稀疏奖励任务。我们的配方专为离线到在线 RL 设置设计，目标是利用离线先验数据集最大化在线学习的样本效率。

有效的探索和样本高效学习仍然是此设置中的核心挑战，因为如何利用离线数据获取好的探索策略并不明显。我们的关键洞察是，**动作块化**——一种在模仿学习中流行的技术，其中预测未来动作序列而非每个时间步的单个动作——可以应用于基于时序差分（TD）的 RL 方法以缓解探索挑战。

Q-chunking 通过直接在"分块"动作空间中运行 RL 来采用动作块化，使智能体能够（1）利用离线数据的时间一致行为进行更有效的在线探索，以及（2）使用无偏 n 步备份进行更稳定高效的 TD 学习。

## 1 引言

强化学习（RL）有望仅基于奖励函数解决任何给定任务。然而，这种简单直接的 RL 问题公式往往不切实际：在复杂环境中，完全从头探索学习有效策略可能极其昂贵。

**核心洞察**：虽然我们可能最终想要一个最优的马尔可夫策略，但探索问题可以用非马尔可夫和时间扩展技能更好地解决，动作块化提供了获取这种技能的简单方便配方。

### 贡献

1. **Q-chunking 配方**：在分块动作空间中运行 RL
2. **QC 算法**：隐式 KL 行为约束的实际实现
3. **QC-FQL**：应用于 FQL 的实例化
4. **理论分析**：无偏 n 步备份的证明

## 2 相关工作

### 离线到在线 RL

利用先验离线数据加速在线学习。简单方法：先离线预训练再在线微调，但往往过于悲观。

### 动作块化

模仿学习中广泛使用，预测未来动作序列。已有 RL 方法通常专注于微调模仿学习预训练的策略。

### 层次 RL

学习时间扩展动作。低级策略与高级策略的双层优化问题。Q-chunking 是 HRL 的特例，低级技能开环执行动作序列。

### 多步潜在空间规划

基于模型的 RL 方法在编码潜在空间中优化短时程动作序列。

## 3 背景

### 离线到在线 RL

**MDP**：(S, A, ρ, T, r, γ)

**目标**：最大化期望折扣累积奖励

### 时序差分和多步返回

**1 步 TD**：
```
V̂_{1-step} = r_t + γ·Q̄_θ(s_{t+1}, a_{t+1})
```

**n 步返回**：
```
V̂_{n-step} = Σ γ^{t'-t} r_{t'} + γ^n·Q̄_θ(s_{t+n}, a_{t+n})
```

**问题**：n 步返回在离策略时有偏

---

## 4 Q-Chunking

### 4.1 时间扩展动作空间上的 Q 学习

**Q-chunking 策略**：π_ψ(a_{t:t+h} | s_t)

**Q-chunking 评论家**：Q_θ(s_t, a_{t:t+h})

**TD 损失**：
```
L(θ) = E[(Q_θ(s_t, a_{t:t+h}) - (Σ γ^{t'} r_{t+t'} + γ^h Q̄_θ(s_{t+h}, a_{t+h:t+2h})))²]
```

**三种 TD 备份对比**：
```
Q(s_t, a_t) ← r_t + γQ(s_{t+1}, a_{t+1})                    (标准 1 步 TD)
Q(s_t, a_t) ← Σ γ^{t'-t} r_{t'} + γ^h Q(s_{t+h}, a_{t+h})  (n 步返回，有偏)
Q(s_t, a_{t:t+h}) ← Σ γ^{t'-t} r_{t'} + γ^h Q(s_{t+h}, a_{t+h:t+2h})  (Q-chunking，无偏)
```

**关键区别**：Q-chunking 的 Q 函数接收完整动作序列，消除了 n 步返回的偏差

### 4.2 时间一致探索的行为约束

**行为约束**：
```
D(π_ψ(a_{t:t+h}|s_t), π_β(a_{t:t+h}|s_t)) ≤ ε
```

**优势**：
- 离线数据常展现非马尔可夫结构
- 时间扩展动作空间的行为约束更好捕获这种结构
- 时间一致动作帮助结构化探索

### 4.3 实际实现

**QC：隐式 KL 行为约束**

**流程**：
1. 从学习的行为策略 f_ξ(·|s) 采样 N 个动作块
2. 选择最大化 Q 函数的动作块：a* = arg max Q(s, a)

**Best-of-N 采样**：隐式施加 KL 约束

**QC-FQL**：应用于 FQL 的实例化

## 5 实验

### OGBench 基准

**5 个长时程稀疏奖励领域**

### 核心结果

- QC 在 5 个领域上聚合性能最强
- 超越 FQL、RLPD、IFQL 等先前最佳方法

### 消融实验

- RLPD-AC（朴素动作块化）：性能差
- QC-RLPD（加行为克隆损失）：性能好
- 高斯策略不足，需要流匹配策略
