# 样本高效的视觉-语言-动作模型强化学习微调

# EXPO-FT: Sample-Efficient Reinforcement Learning Finetuning for Vision-Language-Action Models

**作者**：Perry Dong, Kuo-Han Hung, Tian Gao, Dorsa Sadigh, Chelsea Finn

**机构**：Stanford University

**发表于**：arXiv (cs.RO), 2026

**arXiv ID**：2605.25477

---

## 摘要

视觉-语言-动作（VLA）模型在多样化操作任务上展示了强大的泛化能力，但始终未能达到实际部署所需的可靠性。我们提出了 EXPO-FT，一个用于预训练 VLA 策略的稳定、样本高效 RL 微调系统，弥合了这一差距。现有的 RL 方法要么不利用预训练先验从零训练，要么在微调 VLA 时无法达到实际所需的样本效率和成功率。EXPO-FT 在需要高精度、动态动作和对多样化初始状态鲁棒性的复杂操作任务上进行了验证，包括灯串布线、台球入袋和花瓶插花。我们的系统在所有评估任务上实现了完美性能（30/30 成功），平均仅需 19.1 分钟的在线机器人数据，优于先前的 RL-from-scratch 和 VLA 微调方法。我们发布了开源代码库。

---

## 1 引言

预训练 VLA 模型在多样化操作任务上展示了令人印象深刻的泛化能力。然而，这些模型在实际部署中经常无法达到所需的可靠性水平。强化学习（RL）有望通过试错学习来改进这些策略，但直接对大型 VLA 模型进行 RL 微调面临两个主要挑战：

1. **样本效率**：真实世界数据收集成本高昂
2. **稳定性**：RL 训练可能破坏预训练知识

EXPO-FT 通过以下关键设计解决这些问题：
- 冻结预训练 VLA 作为基策略
- 引入轻量编辑策略进行探索
- 使用 OTF（On-the-Fly）动作选择机制
- 整合人类在环干预

---

## 2 相关工作

### 2.1 VLA 模型

VLA 模型（如 RT-2、OpenVLA、π0）将视觉感知、语言理解和动作生成统一到单一框架中。这些模型通过大规模预训练获得强大泛化能力，但在特定任务上的可靠性仍需提升。

### 2.2 真实世界 RL

真实世界机器人 RL 需要样本高效的方法。SERL 和 HIL-SERL 展示了 RL 在真实世界中的潜力，但从零训练的样本效率仍有提升空间。

### 2.3 VLA 微调

监督微调（SFT）是常见的 VLA 适应方法，但受限于演示数据质量和分布偏移问题。

## 3 方法

### 3.1 问题定义

给定预训练 VLA 策略 π_VLA，目标是通过 RL 微调使其在目标任务上达到高可靠性，同时最小化真实世界数据需求。

### 3.2 EXPO-FT 架构

#### 基策略（Base Policy）

预训练的流匹配 VLA 模型，在 RL 训练期间**完全冻结**：

```
π_VLA: (o_t, l) → {a_1, ..., a_8}  # 采样 8 个随机动作 chunk
```

#### 编辑策略（Edit Policy）

轻量级网络，预测对基动作的修正：

```
π_edit: (o_t, a_t) → â_t  # 修正量，约束在 [-β, β]
```

编辑后的动作：
```
ã_t = a_t + â_t
```

#### OTF 动作选择

从 16 个候选动作（8 个基动作 + 8 个编辑动作）中选择 Q 值最高的：

```
ã* = argmax_{a ∈ {a_i} ∪ {ã_i}} Q_φ(s, a)
```

**关键优势**：
- 不需要对大 VLA 做梯度回传
- Q 网络评估基动作和编辑动作的质量
- 动态选择最优动作

### 3.3 Q 网络架构（Critic）

#### RedQ 风格集成

- 10 个 Q 网络的集成
- 每次随机选 2 个，取最小值作为目标
- 使用层归一化

#### 视觉编码

独立的轻量 ResNet-50（不共享 VLA 编码器）：
- 阶段深度 (3,4,6,3)
- 64 个滤波器
- 输出 512 维图像嵌入

#### 本体感觉编码

- 嵌入到 64 维
- 与视觉特征融合

#### MLP

- 3 个隐藏层，宽度 256

### 3.4 时序扩展动作（Action Chunks）

编辑策略和 Q 函数都操作在动作 chunk（长度 H）上：
- 每次重新规划执行 C ≤ H 步
- 保持现代 VLA 的时序抽象设计

编辑策略损失（扩展到动作 chunk）：

```
L(π_edit) = -E[Q_φ(s_t, a_{t:t+C} + â_{t:t+C}) - α log π_edit(â_{t:t+C}|s_t, a_{t:t+C})]
```

Q 函数损失：

```
L(φ) = E[(r_t + γ Q_{φ'}(s_{t+C}, ã*_{t+C:t+2C}) - Q_φ(s_t, a_{t:t+C}))²]
```

### 3.5 人类在环干预

训练期间，人类操作员可以通过 SpaceMouse 覆盖动作 chunk 中的任意时间步：

```
训练初期：人类频繁干预
    ↓
策略改进：干预频率逐渐降低
    ↓
训练后期：干预降至零
```

纠正数据存入回放缓冲区，提供有针对性的监督信号。

### 3.6 训练流程

```
步骤1：评估零样本 VLA 性能
       如果不足 → 收集演示 + 监督微调至 ~40% 成功率

步骤2：用离线数据初始化回放缓冲区

步骤3：在线 RL 训练
       Learner-Actor 解耦架构
       Learner：处理 VLA 训练/推理
       Actor：处理环境交互
       支持同步/异步通信
```

### 3.7 奖励设计

稀疏二元奖励（0 或 1），基于规则的分类器，准确率 >95%：

- 糖果检测：像素颜色匹配
- 方块拾取：高度阈值
- 任务特定检测器

### 3.8 优化超参数

| 参数 | 值 |
|------|-----|
| 优化器 | Adam |
| 学习率 | 3×10⁻⁴ |
| 折扣因子 γ | 0.99 |
| 初始温度 α₀ | 1.0 |
| Critic 目标 τ_Q | 5×10⁻³ |
| 策略目标 τ_π | 10⁻³ |
| 批大小 | 64 |
| UTD 比率 | 20 |

## 4 实验

### 4.1 实验任务

| 任务 | 描述 | 难度 |
|------|------|------|
| String Light Routing | 将灯串穿过挂钩 | 高精度、柔性物体 |
| Pool Ball Pocketing | 将台球打入袋 | 动态操作 |
| Flower Insertion | 将花插入花瓶 | 高精度装配 |
| Plug Insertion | 将插头插入插座 | 精密装配 |

### 4.2 核心结果

| 指标 | 结果 |
|------|------|
| 成功率 | 30/30（100%）所有任务 |
| 平均训练时间 | 19.1 分钟在线数据 |
| 对比基线 | 优于 RL-from-scratch 和 VLA 微调方法 |

### 4.3 消融实验

| 方法 | 成功率 |
|------|--------|
| RL from scratch | 低 |
| VLA 直接微调 | 中 |
| EXPO-FT | 100% |

### 4.4 鲁棒性

训练后的策略展示了出色的零样本鲁棒性：
- 适应多样化初始状态
- 处理外部干扰
- 从错误中恢复

---

## 5 结论

我们展示了 EXPO-FT，一个用于预训练 VLA 策略的稳定、样本高效 RL 微调系统。通过冻结基 VLA、引入轻量编辑策略和 OTF 动作选择机制，我们的系统在所有评估任务上实现了完美性能（30/30 成功），平均仅需 19.1 分钟的在线机器人数据。我们的方法优于先前的 RL-from-scratch 和 VLA 微调方法，为 VLA 模型的实际部署提供了一条可行路径。
