# 用于微调行为克隆策略的残差离策略 RL

# Residual Off-Policy RL for Finetuning Behavior Cloning Policies

**作者**：Lars Ankile, Zhenyu Jiang, Rocky Duan, Guanya Shi, Pieter Abbeel, Anusha Nagabandi

**机构**：Amazon FAR (Frontier AI & Robotics), Stanford, CMU, UC Berkeley

**arXiv**: 2509.19301v2 [cs.RO] 2025年9月25日

---

## 摘要

行为克隆（BC）的最新进展已经实现了令人印象深刻的视觉运动控制策略。然而，这些方法受限于人类演示的质量、数据收集所需的手动工作以及离线数据的收益递减。相比之下，强化学习（RL）通过自主与环境交互训练智能体，并在各个领域展示了显著成功。尽管如此，直接在真实世界机器人上训练 RL 策略仍然具有挑战性，原因是样本低效、安全问题以及从稀疏奖励学习长时程任务的困难，特别是对于高自由度（DoF）系统。

我们提出了一种通过残差学习框架结合 BC 和 RL 优势的方法。我们的方法将 BC 策略视为黑盒基础，并通过样本高效的离策略 RL 学习轻量级逐步残差校正。我们展示了我们的方法仅需要稀疏二元奖励信号，可以在仿真和真实世界中有效改进高自由度系统的操作策略。据我们所知，我们首次成功展示了在具有灵巧手的人形机器人上进行真实世界 RL 训练。

## 1 引言

使机器人能够直接在部署环境中学习和改进仍然是机器人学的基本挑战。最近，在使用行为克隆（BC）从人类演示训练视觉运动控制策略方面取得了显著进展。然而，这种成功需要大量基础设施以及数小时繁琐的手动数据收集。

强化学习（RL）提供了一种互补范式：智能体通过试错在环境中自主学习。然而，强大的 RL 性能通常需要大量在线交互数据，因此其应用主要在仿真中。

改进 BC 策略的自然方向是利用在线 RL，结合两者的优势：BC 策略提供可以正则化 RL 过程探索的强大先验，而在线 RL 通过从环境交互中学习来增强策略性能。

### 核心思想

**残差 RL** 是一种简单而强大的方法：RL 不学习完整策略，仅在固定基础控制器上学习校正项。本文提出离策略残差微调（ResFiT）方法。

### 贡献

1. **ResFiT 框架**：离策略残差微调方法
2. **首个真实世界 RL**：在灵巧手人形机器人上成功训练
3. **高自由度系统**：扩展到 29 自由度双臂系统
4. **设计决策分析**：详细消融研究

## 2 相关工作

### 行为克隆

BC 在各种设置中展示了成功，从自回归下一 token 预测到扩散策略，再到直接输出机器人动作的大型 Transformer。尽管有这些进展，BC 有根本性局限。

### 使用 RL 微调 BC 策略

RL 微调已被探索作为增强 BC 策略的方法。然而，直接用 RL 微调现代 BC 架构存在重大挑战。IBRL 训练模仿策略然后用于提议动作。PA-RL 学习 Q 函数优化动作而非策略。DSRL 在冻结基础扩散策略的潜在噪声空间上运行 RL。

### 残差 RL 和样本效率

残差 RL 提供了在固定基础策略上学习校正的方法。ResiP 结合现代动作块 BC 策略与单步 RL 残差策略，但其在策略 RL 算法样本复杂度过高。Policy Decorator 和 EXPO 展示了离策略 RL 可以更高效训练残差策略，但仅限于仿真和单臂任务。

## 3 方法

### 3A. 基础策略：带动作块的行为克隆

**基础策略训练**：
```
min_ψ -Σ log π_ψ(a_{t:t+k} | o_t)
```

预测 k 步未来动作序列，提高性能并缓解复合误差。

### 3B. 离策略残差 RL 微调

**关键设计**：
- 冻结基础策略 π_base
- 训练残差策略 π_res 学习校正
- 完整动作：a_t = a_base_t + a_res_t
- 评论家预测完整动作的价值

**残差策略参数化**：
```
Q_ϕ(s_t, a_base_t + π_θ(s_t, a_base_t))
π_θ(s_t, a_base_t)
```

**评论家损失**：
```
L(ϕ) = E[(Q_ϕ(s_t, a_t) - (r_t + γ(1-d)Q_ϕ(s_{t+1}, a_base_{t+1} + π_θ(s_{t+1}, a_base_{t+1}))))²]
```

**策略损失**：
```
L(θ) = -E[Q_ϕ(s_t, a_base_t + π_θ(s_t, a_base_t))]
```

### 3C. 关键设计决策

1. **高更新-数据比（UTD > 1）**：提高样本效率
2. **n 步回报（n=3）**：帮助长时程和稀疏奖励任务
3. **层归一化**：缓解灾难性高估
4. **延迟 Actor 更新**：每 2-8 个评论家更新一次
5. **Polyak 平均**：τ = 0.005 更新目标网络
6. **目标策略平滑**：向动作添加噪声
7. **随机集成双 Q 学习**：减少高估偏差
8. **浅层 ViT 编码器**：带 DrQ 风格随机移位增强
9. **对称采样**：50% 离线数据 + 50% 在线数据

## 5 结果

### 5A. 仿真结果

**样本效率提升**：
- 离策略残差 RL vs PPO：200 倍样本效率提升
- ResFiT 在 200k 步收敛 vs PPO 在 40M 步

**各任务性能**：
- Can：所有方法在 ~150k 步达到高成功率
- Square：仅 ResFiT 和 Tuned RLPD 达到 >90%
- BoxCleanup/CanSort/Coffee：ResFiT 高效收敛到近完美

**消融实验**：
- 无层归一化：性能下降
- 无离线缓冲区：性能下降
- Filtered BC：稳定但改进有限
- IBRL：在高自由度任务上表现不佳

**UTD 和 n-step 分析**：
- n-step > 1 对稀疏奖励任务重要
- UTD > 1 提供明显收益，但极高 UTD 收益递减
- UTD = 4 捕获大部分收益同时保持稳定

### 5B. 真实世界结果

| 任务 | 基础策略 | ResFiT | RL 数据 |
|------|----------|--------|---------|
| WoollyBallPnP | 14% | **64%** | 134 episodes (~15 分钟) |
| PackageHandover | 23% | **64%** | 343 episodes (~76 分钟) |

**首次成功**：在具有两只五指手的双臂灵巧操作人形机器人上进行真实世界 RL 训练

## 6 讨论

### 关键洞察

1. **解耦预训练和微调**：动作块和大策略改进 BC 性能，但为 RL 创建了难以处理的高维动作空间
2. **基础策略的双重作用**：
   - 隐式安全约束：无此正则化的策略收敛到更快但更激进的行为
   - 强大探索先验：即使在高自由度空间也能从稀疏奖励进行 RL

### 局限性

1. **基础策略约束**：学习的行为可能受限于基础策略附近
2. **无法发现根本不同策略**：不能超越基础策略编码的技能
3. **人类监督**：需要重置和奖励标注

### 未来方向

- 放松冻结基础约束
- 蒸馏更精确行为回基础策略
- 多任务设置中的任务特定残差改进
- 自动重置和成功检测
