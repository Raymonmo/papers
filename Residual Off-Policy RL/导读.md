## 🎯 一句话总结

**提出 ResFiT，一种基于离策略 RL 的残差微调方法，将 BC 策略视为黑盒，通过轻量级逐步残差校正实现高效在线 RL 微调，首次在具有灵巧手的人形机器人上成功进行真实世界 RL 训练。**

---

## 🔍 研究背景与动机

### 核心问题

1. **BC 的局限**：
   - 依赖人类演示质量
   - 数据收集需要大量手动工作
   - 离线数据存在收益递减

2. **直接 RL 的挑战**：
   - 样本低效
   - 安全问题
   - 稀疏奖励下长时程任务学习困难
   - 高自由度系统尤其困难

3. **现有残差 RL 的局限**：
   - 仅在仿真中学习
   - 简单或受限设置
   - 缺乏高自由度真实世界应用

### 核心思想

将 BC 策略视为黑盒，通过离策略 RL 学习轻量级逐步残差校正，避免直接优化庞大的基础策略。

---

## 💡 核心贡献

1. **ResFiT 框架**：离策略残差微调方法，结合 BC 和 RL 优势
2. **首个真实世界 RL**：在具有灵巧手的人形机器人上成功训练
3. **高自由度系统**：扩展到 29 自由度双臂灵巧操作系统
4. **设计决策分析**：详细消融研究验证每个设计选择

## 📖 方法详解

### 3A. 基础策略：带动作块的行为克隆

**基础策略训练**：
```
min_ψ -Σ log π_ψ(a_{t:t+k} | o_t)
```

**动作块的优势**：
- 提高性能
- 通过有效减少任务时域来缓解模仿学习中的复合误差

### 3B. 离策略残差 RL 微调

**关键设计**：
- 冻结基础策略 π_base
- 训练残差策略 π_res 学习校正
- 完整动作：a_t = a_base_t + a_res_t
- 评论家预测完整动作的价值

**残差策略参数化**：
```
Q_ϕ(s_t, a_base_t + π_θ(s_t, a_base_t))
π_θ(s_t, a_base_t)
```

**贝尔曼方程**：
```
Q*(s_t, a_t) = E[r_t + γ max_{a'} Q*(s_{t+1}, a')]
```

**评论家损失**：
```
L(ϕ) = E[(Q_ϕ(s_t, a_t) - (r_t + γ(1-d)Q_ϕ(s_{t+1}, a_base_{t+1} + π_θ(s_{t+1}, a_base_{t+1}))))²]
```

**策略损失**：
```
L(θ) = -E[Q_ϕ(s_t, a_base_t + π_θ(s_t, a_base_t))]
```

### 3C. 关键设计决策

1. **高更新-数据比（UTD > 1）**：提高样本效率
2. **n 步回报（n=3）**：帮助长时程和稀疏奖励任务
3. **层归一化**：缓解灾难性高估
4. **延迟 Actor 更新**：每 2-8 个评论家更新一次
5. **Polyak 平均**：τ = 0.005 更新目标网络
6. **目标策略平滑**：向动作添加噪声
7. **随机集成双 Q 学习**：减少高估偏差
8. **浅层 ViT 编码器**：带 DrQ 风格随机移位增强
9. **对称采样**：50% 离线数据 + 50% 在线数据

## 📊 实验设置与主要结果

### 仿真任务

| 任务 | 机器人 | 自由度 | 描述 |
|------|--------|--------|------|
| Can | Franka | 7 | 罐头操作 |
| Square | Franka | 7 | 方块操作 |
| BoxCleanup | 双 Franka | 24 | 盒子清理 |
| CanSort | GR1 人形 | 24 | 罐头分类 |
| Coffee | GR1 人形 | 24 | 咖啡制作 |

### 仿真结果

**样本效率提升**：
- 离策略残差 RL vs PPO：200 倍样本效率提升
- ResFiT 在 200k 步收敛 vs PPO 在 40M 步

**各任务性能**：
- Can：所有方法在 ~150k 步达到高成功率
- Square：仅 ResFiT 和 Tuned RLPD 达到 >90%
- BoxCleanup/CanSort/Coffee：ResFiT 高效收敛到近完美

**消融实验**：
- 无层归一化：性能下降
- 无离线缓冲区：性能下降
- Filtered BC：稳定但改进有限
- IBRL：在高自由度任务上表现不佳

### 真实世界任务

| 任务 | 描述 | 基础策略 | ResFiT |
|------|------|----------|--------|
| WoollyBallPnP | 右手抓取毛球放入箱中 | 14% | **64%** |
| PackageHandover | 双臂包裹交接 | 23% | **64%** |

**训练数据**：
- WoollyBallPnP：134 个 RL episode（~15 分钟）
- PackageHandover：343 个 RL episode（~76 分钟）

**首次成功**：在具有两只五指手的双臂灵巧操作人形机器人上进行真实世界 RL 训练

## 🔄 与相关工作的对比

| 方法 | 特点 | 局限 |
|------|------|------|
| BC | 模仿学习 | 受限于演示质量 |
| RLPD | 离策略 RL | 直接学习完整策略困难 |
| IBRL | BC 提议动作 | 高自由度任务效果差 |
| Filtered BC | 在线 BC 微调 | 精度改进有限 |
| PPO 残差 | 在策略残差 RL | 样本效率低（200x 差距） |
| Policy Decorator | 块级残差 | 仅仿真，单臂 |
| DSRL | 扩散噪声空间 | 仅仿真 |
| **ResFiT** | **逐步残差 + 离策略** | **受限于基础策略** |

## ⚠️ 局限性与未来方向

### 局限性
1. **基础策略约束**：学习的行为可能受限于基础策略附近
2. **无法发现根本不同策略**：不能超越基础策略编码的技能
3. **人类监督**：需要重置和奖励标注

### 未来方向
- 放松冻结基础约束
- 蒸馏更精确行为回基础策略
- 多任务设置中的任务特定残差改进
- 自动重置和成功检测

## 💬 个人评价与启发

### 优点
1. **实用性强**：首次在灵巧手人形机器人上成功 RL
2. **设计优雅**：黑盒残差方法，与基础策略解耦
3. **样本高效**：离策略方法 + 多项优化技术
4. **深入消融**：详细验证每个设计选择
5. **真实世界验证**：29 自由度双臂系统

### 启发
1. **残差学习的优势**：避免直接优化庞大模型
2. **基础策略的双重作用**：初始化 + 隐式安全约束
3. **离策略方法的价值**：比在策略方法样本效率高 200 倍
4. **高自由度 RL 的可行性**：通过合适设计可以扩展
