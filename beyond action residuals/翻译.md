# 超越动作残差：通过瓶颈潜在强化学习实现真实世界机器人策略引导

# Beyond Action Residuals: Real-World Robot Policy Steering via Bottleneck Latent Reinforcement Learning

**作者**：Dongjie Yu, Kun Lei, Zhennan Jiang, Jia Pan, Huazhe Xu

**机构**：University of Hong Kong, Shanghai Qizhi Institute, Shanghai Jiao Tong University, Chinese Academy of Sciences, Tsinghua University

**arXiv**: 2605.19919v1 [cs.RO] 2026年5月19日

## 摘要

预训练模仿策略已成为机器人操作的强大基础，但它们通常需要在线改进以克服执行误差、有限数据集覆盖和部署不匹配。因此，一个核心问题是强化学习（RL）在离线预训练后应如何适应策略。现有的轻量级方法通常直接在动作空间中应用残差校正，但这常常导致嘈杂且结构不良的探索。

在这项工作中，我们提出了 **Z 扰动强化学习（ZPRL）**，一种通过紧凑瓶颈潜在空间（而非策略权重或输出动作）引导预训练策略的方法。在离线训练期间，我们用即插即用的变分信息瓶颈（VIB）模块增强策略，从观测嵌入中提取任务相关潜在接口。在在线微调期间，基础策略被冻结，RL 仅在此潜在空间上学习残差扰动，其解码表示条件化冻结动作生成器。

我们在流匹配策略上实例化 ZPRL，在八个仿真任务和四个真实世界任务上进行评估。在多样化操作设置中，ZPRL 相比强后训练基线提高了样本效率和最终性能。在真实世界中，ZPRL 在四个任务上将平均成功率比模仿基础策略提高了 33.7%，同时产生比动作残差对应物更平滑的探索行为。

## 1 引言

在离线数据集上的模仿学习（IL）已成为构建机器人操作策略的日益流行方法。最近的策略类别，包括 Transformer 和生成式感觉运动策略（如 VAE 或扩散和流模型），可以从数据中以显著保真度再现复杂行为。

然而，强大的离线预训练并不能消除在线改进的需求：在真实世界部署中，策略可能由于执行误差、演示的任务覆盖不足等原因而无法完成任务。

### 核心问题

**RL 应该如何在离线预训练后适应现代模仿学习策略？**

**现有方法的局限**：
- **全策略微调**：随着策略规模增长越来越昂贵
- **动作空间残差**：探索在低级别，容易导致振荡或不稳定行为

### 核心洞察

**有意义的行为很少从无结构的振荡中产生**——需要的不仅是不同的动作，而是与预训练策略风格一致的不同动作。

### 贡献

1. **ZPRL 框架**：通过瓶颈潜在空间引导预训练策略的轻量级后训练框架
2. **VIB 模块**：即插即用的变分信息瓶颈，提取任务相关潜在表示
3. **结构化探索**：在紧凑潜在空间中进行有结构的探索
4. **真实世界验证**：在四个真实世界任务中验证有效性

## 2 相关工作

### 机器人操作中的模仿学习

从低维技能表示发展到大规模视觉运动行为建模。尽管性能令人印象深刻，这些策略仍然受限于离线数据的覆盖和质量。

### 用于适应预训练策略的强化学习

**权重空间适应**：端到端微调整个策略。

**动作空间适应**：残差 RL 方法在经典控制器或冻结模仿策略上学习校正动作。轻量级但探索在低级别。

**潜在空间适应**：最近的工作将 RL 干预从动作空间移到内部潜在空间，如初始扩散噪声。

### ZPRL 的独特之处

使用紧凑瓶颈潜在作为引导接口，结合轻量级适应与任务对齐低维潜在空间。

## 3 预备知识和问题形式化

### 3A. 机器人操作作为观测条件决策过程

将机器人操作建模为 episodic MDP，使用稀疏二元奖励。

### 3B. 流匹配基础策略

**流匹配策略训练**：
```
L_IL(ϕ) = E[||v_ϕ(a_k, k, c) - (w - a)||²]
```

**动作生成**：
```
a_{k_m} = a_{k_{m+1}} - (k_{m+1} - k_m) · v_ϕ(a_{k_{m+1}}, k_{m+1}, c)
```

### 3C. RL 后训练设置

**三种适应接口**：
1. **权重空间**：直接更新策略参数
2. **动作空间**：a ~ π_base(·|c) + λ·π_on(·|o)
3. **潜在空间（ZPRL）**：在瓶颈潜在空间中进行扰动

## 4 用 Z 扰动强化学习引导策略

### 4A. 瓶颈任务潜在用于策略条件化

**信息瓶颈目标**：
```
min -I(z; a) + β·I(z; c)
```

**VIB 代理目标**：
```
L_VIB(φ) = E[-E_{z~p_φ(·|c)} log q(a|g_φ(z)) + β·KL(p_φ(z|c) || r(z))]
```

**整体离线损失**：
```
L_off(ϕ, φ) = L_IL(ϕ) + L_VIB(φ)
```

**关键设计**：VIB 作为即插即用辅助路径，梯度被阻止更新基础编码器和流生成器。

**瓶颈的作用**：与 SOE 不同，ZPRL 中学习的潜在 z 包含 RL 后训练的引导潜力。

### 4B. 瓶颈潜在上的 RL 扰动

**扰动公式**：
```
z̃ = z + λ·Δz, Δz ~ π_on(·|c, z)
```

**最终动作**：
```
a_on ~ π_base(·|g_φ(z̃))
```

**优势**：
- 在低维空间（16-32 维）中操作，而非高维动作空间（可达 100 维）
- 通过冻结解码器和流生成器产生结构化动作
- 扰动作用于策略条件而非直接作用于生成动作

**评论家定义**：
```
r̂(c, z̃) := r(s, a_on), Q̂(c, z̃) := Q(s, a_on)
```

### 4C. 在线 RL 目标和设计选择

**Actor 损失**：
```
L_π(θ) = E[α·log π_θ(Δz|c,z) - Q_ψ(c, z̃)]
```

**Critic 损失**：
```
L_Q(ψ) = E[(Q_ψ(c, z̃) - y)²]
```

**设计选择**：
- 扰动尺度 λ：控制探索-稳定性权衡
- 更新-数据比（UTD）：仿真用 1，真实世界用 2 或 5
- 不使用动作空间评论家：避免迭代去噪开销

## 5 实验

### 5A. 仿真中的在线 RL 微调

**三个基准，八个任务**：
- Robomimic：can, square, transport
- Adroit：door, hammer, pen（灵巧手）
- Metaworld：box-close, push-wall

**基线方法**：DPPO, ReinFlow, Policy Decorator, DSRL

**结果**：ZPRL 在所有八个任务上一致达到强最终性能。

### 5B. 瓶颈潜在接口如何塑造在线 RL

**消融实验**：

1. **瓶颈潜在 vs 直接观测嵌入引导**：ZPRL 优于直接在观测嵌入上学习残差

2. **扰动尺度 λ**：λ = 0.20 提供最佳平衡（1.16M 步达到 SR=0.9，最终 SR=0.98）

3. **瓶颈维度**：dim(z) = 8-32 都达到强性能，dim(z) = 32 最佳

4. **离线数据量**：更少演示导致更多步骤和更低最终性能，但即使只有 25 个演示仍然有效

**关键洞察**：
- 增益不仅来自降维
- 任务相关空间比任意中间特征更重要
- 紧凑但不过度限制的接口最有效

### 5C. 平滑和结构化动作生成

| 指标 | Policy Decorator | ZPRL | 改善 |
|------|------------------|------|------|
| Vel_EE (m/s) | 0.31 | 0.22 | -29% |
| Acc_EE (m/s²) | 9.60 | 5.79 | -39% |

**ZPRL 产生更平滑、更结构化的行为**，减少高频振荡。

### 5D. 真实世界实验

**四个真实世界任务**：
- 平均成功率提升 33.7%
- 比动作残差方法产生更平滑的探索行为
- 更低的末端执行器速度和加速度

### 5E. 讨论

**局限性**：
1. VIB 模块需要额外离线训练
2. 扰动尺度需要为每个任务调整
3. 受限于预训练策略的能力

**未来方向**：自适应扰动尺度、更多策略架构、多任务瓶颈潜在共享。

---

## 6 结论

我们提出了 ZPRL，一种通过紧凑瓶颈潜在空间引导预训练策略的轻量级后训练框架。通过即插即用 VIB 模块提取任务相关潜在表示，在线 RL 在此紧凑空间中学习残差扰动，产生比动作残差方法更平滑、更结构化的行为。在仿真和真实世界实验中，ZPRL 提高了样本效率和最终性能。
