# RLDG：通过强化学习进行机器人通用策略蒸馏

# RLDG: Robotic Generalist Policy Distillation via Reinforcement Learning

**作者**：Charles Xu, Qiyang Li, Jianlan Luo, Sergey Levine

**机构**：UC Berkeley

**arXiv**: 2412.09858v1 [cs.RO] 2024年12月13日

**项目网站**：https://generalist-distillation.github.io/

---

## 摘要

机器人基础模型的最新进展使得能够适应多样化任务的通用策略的开发成为可能。虽然这些模型展示了令人印象深刻的灵活性，但它们的性能严重依赖于训练数据的质量。

在这项工作中，我们提出**强化学习蒸馏通用策略（RLDG）**，一种利用强化学习生成高质量训练数据来微调通用策略的方法。通过在精密操作任务（如连接器插入和装配）上的大量真实世界实验，我们展示了使用 RL 生成数据训练的通用策略始终优于使用人类演示训练的策略，成功率高达 40%，同时更好地泛化到新任务。

我们还提供了详细分析，揭示这种性能源于优化的动作分布和改进的状态覆盖。我们的结果表明，将任务特定 RL 与通用策略蒸馏相结合，为开发更强大、更高效的机器人操作系统提供了有前景的方法。

## 1 引言

机器人基础模型的最新进展展示了理解和执行多样化操作技能的令人印象深刻的能力。通过利用互联网规模预训练和与机器人动作的对齐，这些模型可以实现零样本和少样本泛化。

**核心挑战**：部署这些模型通常需要使用任务特定数据微调。人类遥操作是常见数据来源，但人类演示包含执行质量和风格的不一致。

**RLDG 的核心洞察**：RL 代理可以通过奖励最大化自主生成高质量轨迹，比人类演示更适合微调通用策略。

### 贡献

1. **RLDG 方法**：利用 RL 生成高质量训练数据微调基础模型
2. **性能提升**：比人类演示数据高 30-50% 成功率
3. **数据效率**：所需数据量减少 6-10 倍
4. **灵活组合**：可在多阶段任务中与人类演示灵活组合

---

## 2 相关工作

### 机器人基础模型

利用大规模视觉-语言预训练，然后在机器人上演示上微调。展示广泛泛化，但在精密操作任务上挣扎。

### 机器人操作的 RL

成功应用于精密插入、多阶段装配、灵巧手内操作。优势：发现最优动作分布。局限：难以泛化超出训练分布。

### 策略蒸馏和知识转移

将多个专门策略蒸馏为单一通用策略。RLDG 的创新：（1）蒸馏到预训练基础模型比从头训练更好；（2）RL 生成数据微调优于人类演示。

## 3 强化学习蒸馏通用策略

### 3.1 在线 RL 训练

**MDP 公式**：
```
J(π) = E[Σ γ^t R(s_t, a_t)]
```

**使用 HIL-SERL**：样本高效，从像素输入学习精密操作技能

### 3.2 经验收集

**流程**：
1. 训练 RL 专家策略
2. 推出收敛策略收集数据
3. 构建平衡的微调数据集

**灵活性**：
- 多个 RL 策略的数据可混合
- RL 数据可与人类演示组合

### 3.3 通用策略微调

**监督学习目标**：
```
L(θ) = -E[log π_θ(a_t | s_t)]
```

**OpenVLA 微调**：
- 7B 参数 VLA 模型
- 使用 LoRA 参数高效微调
- 自回归离散化动作

**Octo 微调**：
- 开源通用机器人策略
- 扩散头预测连续动作
- 直接微调整个网络

## 4 实验与结果

### 4.1 实验设置和任务

**机器人设置**：Franka Emika Panda 机械臂 + 平行夹爪 + RealSense D405 腕部相机

**4 个真实世界任务**：

| 任务 | 描述 | 挑战 |
|------|------|------|
| **Connector Insertion** | 电子连接器插入 | 亚毫米精度，接触动力学 |
| **Pick and Place** | 抓取放置 | 视觉泛化 |
| **FMB Insertion** | FMB 基准插入 | ±1.5mm 公差 |
| **FMB Assembly** | FMB 多阶段装配 | 精密插入 + 抓取 |

### 4.2 RLDG vs 传统微调

**RLDG vs 人类演示微调**：

| 任务 | OpenVLA 提升 | Octo 提升 |
|------|-------------|-----------|
| FMB Insertion | +33% | +10% |
| Connector Insertion | +23% | +28% |
| **平均** | **+30%** | **+20%** |

**泛化能力**：
- 未见场景：平均 +50% 成功率
- 未见连接器：显著提升

**数据效率**：
- 达到相同性能需要 6-10 倍更少数据
- 精密任务可达 100% 成功率（人类演示上限 90%）

---

## 5 结论

我们提出了 RLDG，一种利用 RL 生成高质量训练数据微调通用机器人策略的简单有效方法。通过在精密操作任务上的大量实验，我们展示了 RLDG 比人类演示数据训练的策略高 30-50% 成功率，所需数据量减少 6-10 倍。
