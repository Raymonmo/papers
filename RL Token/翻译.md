## 摘要

视觉-语言-动作（VLA）模型可以"开箱即用"地学习执行多样化操作技能，但实现真实世界任务所需的精度和速度需要进一步微调——例如通过强化学习（RL）。我们引入了一种轻量级方法，仅使用几小时的真实世界实践即可实现预训练 VLA 的样本高效在线 RL 微调。我们（1）适应 VLA 以暴露一个"RL token"，这是一个紧凑的读出表示，保留任务相关的预训练知识，同时作为在线 RL 的高效接口；（2）在这个 RL token 上训练小型 actor-critic 网络来精炼动作，同时将学习的策略锚定到 VLA。使用 RL token 的在线 RL（RLT）使得能够快速高效地微调甚至大型 VLA。在四个真实机器人任务（螺丝安装、扎带紧固、充电器插入和以太网插入）中，RLT 将任务最难部分的速度提升高达 3 倍，并在几分钟到几小时的实践中显著提高成功率。

## 1 引言

通用视觉-语言-动作（VLA）模型可以从数据中学习广泛的多样化操作技能。然而它们常常在执行的最后毫米级精度上挣扎：动作可能缓慢，成功完成可能需要暂停重试，精密任务关键阶段的小误差可能累积成失败。

解决这个挑战的自然方式是使用强化学习（RL）微调 VLA。通过在目标任务上练习，RL 可以精确地改进对成功最关键的任务阶段。

但真实世界机器人在严格预算下运行：每个 episode 花费时间，每次失败消耗精力和磨损，有意义的适应通常必须在几小时实践内发生。

### 核心问题

**如何在利用 VLA 泛化能力的同时，实现轻量级在线 RL 的速度和样本效率？**

### 贡献

1. **RL Token 机制**：训练编码器-解码器 Transformer 从 VLA 内部特征中提取紧凑的 RL token
2. **分块动作优化**：在 VLA 原生动作接口上操作，缩短信用分配问题
3. **参考动作正则化**：RL 策略条件化于 VLA 参考动作，将在线 RL 变为局部行为精炼

## 2 相关工作

### 视觉-语言-动作模型

行为克隆从大型演示数据集训练通用机器人操作策略。关键成分包括动作块化和表达性输出分布（扩散或自回归生成）。VLA 模型导入大规模网络先验知识到闭环机器人策略。

### 真实世界强化学习

离策略 actor-critic 方法通过重用回放缓冲区中的转移来提高样本效率。SERL 和 RL100 等系统展示了离策略方法可以在几小时内解决接触丰富操作任务。

### VLA 模型的 RL 微调

**全模型更新**：RECAP 使用优势条件策略提取训练整个 π*0.6 模型。PPO 变体应用于 VLA 微调。

**轻量级方法**：ConRFT 冻结 VLA 编码器微调动作头。Policy Decorator 学习残差策略。PLD 预训练评论家然后学习残差策略。GR-RL 在扩散噪声空间中操作。

**RLT 的独特之处**：
- 引入 RL token——压缩 VLA 内部嵌入的紧凑读出表示
- 在分块动作上操作，与 VLA 原生接口对齐
- 直接条件化于 VLA 参考动作块并正则化

## 3 预备知识

### 视觉-语言-动作模型

典型 VLA 包含两个组件：
- **VLM 骨干**：编码多模态输入到共享 token 序列
- **动作专家**：基于扩散的模块，通过迭代去噪生成连续动作

基于 π0.6 模型，给定最多四个相机图像、语言指令和本体感觉状态，产生动作序列（动作块）：ã_{t:t+H-1} ∈ R^{H×d}，H=50 个动作对应 1 秒控制。

### 强化学习和 Actor-Critic 方法

将机器人控制建模为 MDP (S, A, p, r, γ)。使用稀疏二元奖励。策略和评论家在动作块上操作。

## 4 从 RL Token 进行强化学习

### 4A. 使 VLA 暴露 RL 接口

**目标**：将 VLA 表示压缩为适合轻量级 RL 的紧凑嵌入

**RL Token 提取**：
1. 获取 VLA 最终层 token 嵌入：z = f(s, ℓ; θ_vla)
2. 追加学习的 RL token 嵌入：e_rl = e_ϕ(<rl>)
3. 通过轻量级编码器 Transformer 处理：z_rl = g_ϕ([z_{1:M}, e_rl])_{M+1}
4. 解码器 Transformer 自回归重建原始嵌入

**重建目标**：
```
L_ro = E_D[Σ_i ||h_ϕ(d_ϕ([z_rl, z̄_{1:i-1}]))_i - z̄_i||²]
```

**训练**：在小型任务特定演示数据集上训练，VLA 相对于 L_ro 冻结。可选择与 VLA 监督微调结合。

### 4B. 在线 RL 精炼 VLA 动作块

**评论家训练**：
```
L_Q = E[(Q̂ - Q_ψ(x, a_{1:C}))²]
```

**RL 策略训练**：
```
L_π(θ) = E[-Q_ψ(x, a_{1:C}) + β||a_{1:C} - ã_{1:C}||²]
```

**关键设计**：
- 策略条件化于 VLA 参考动作块 ã_{1:C}
- 正则化项控制策略接近 VLA 参考
- 将在线 RL 变为 VLA 动作分布周围的局部动作编辑

**参考动作 Dropout**：
- 随机将部分 batch 中的参考块替换为零
- 防止策略简单复制参考动作
- 迫使策略保持独立的动作生成路径

## 5 完整系统

**预热**：使用 VLA 参考策略推出 N_warm 步填充回放缓冲区

**推出**：每个动作块边界，VLA 产生参考块，RL 策略输出精炼动作块。人类可选择干预。

**子采样动作块**：存储中间步到回放缓冲区，提高数据效率

**更新**：异步执行推出和学习，使用高更新-数据比（5:1）

**关键阶段聚焦**：RLT 应用于改进每个任务的关键阶段——对应需要高精度的最难部分，让基础 VLA 执行较简单的部分。

## 6 真实世界实验

### 4 个精密操作任务

| 任务 | 描述 | 挑战 |
|------|------|------|
| **螺丝安装** | 使用电动螺丝刀将 M3 螺旋入螺纹座 | 亚毫米对准，视觉线索受限 |
| **扎带紧固** | 将扎带尾穿过狭窄锁槽 | 双臂协调柔性物体，紧密公差 |
| **充电器插入** | 将充电器插入端口 | 精密对准 |
| **以太网插入** | 将以太网线插入端口 | 精密对准 |

### 核心结果

**成功率提升**：
- 螺丝安装：20% → 65%
- 其他任务：显著提升

**执行速度提升**：
- 最难部分：最高 3 倍提速
- 部分任务：超越人类遥操作速度

**训练时间**：几分钟到几小时的在线训练

### 消融实验

**关键组件贡献**：
1. RL Token：提供紧凑有效的状态表示
2. 分块动作预测：缩短信用分配问题
3. 策略正则化：稳定学习，防止偏离 VLA
4. 参考动作传递：保留 VLA 多模态动作分布信息

---

## 7 结论

我们提出了 RLT，一种通过在 VLA 中引入紧凑"RL token"表示实现轻量级在线 RL 微调的方法。RLT 将在线 RL 变为 VLA 动作分布周围的局部行为精炼，而非无约束搜索。在四个真实世界精密操作任务中，RLT 在几分钟到几小时内显著提升成功率和执行速度。
